Cleaning up the patient identifier database

When you have finished with the Healthcare: Medical devices to EMR pattern, you can remove the database, its ODBC definition, and the user identifier and password values that are associated with the data source name.

Complete the following tasks to clean up the database environment:

  1. Remove the database and ODBC definition.

    Follow the instructions that are relevant to your database manager:

    • Clean up DB2
    • Clean up Oracle
  2. Remove the user identifier and password that are associated with the data source name PATIENTSDSN.

    Open a command console window in which you can run IBM Integration Bus commands. Enter the following command to remove the user identifier and password: 

    mqsisetdbparms integration_node -n PATIENTSDSN -d
    3. where integration_node is the name of your integration node.
