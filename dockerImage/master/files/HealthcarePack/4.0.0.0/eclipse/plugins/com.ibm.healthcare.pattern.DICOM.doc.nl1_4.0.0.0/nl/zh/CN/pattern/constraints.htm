有关使用“Healthcare：Web 服务到 DICOM”模式的约束

使用“Healthcare：Web 服务到 DICOM”模式时，存在多项约束。

您仅在满足以下所有约束的情况下才能使用“Healthcare：Web 服务到 DICOM”模式：

  • 请求的接收形式必须是通过 HTTP 传送的 SOAP 消息。
  • 发出请求的应用程序必须支持同步请求响应交互样式。
  • 发出请求的应用程序必须支持“Healthcare：Web 服务到 DICOM”模式所定义的搜索条件 XML 格式和结果 XML 格式。
  • 如果要将 DICOM 实例移至目标应用程序，那么 DICOM PACS 必须支持 C-MOVE 命令。
  • DICOM PACS 必须支持 C-FIND 命令的患者或研究查询根。
  • 如果需要进行日志记录或错误处理，那么假定“Healthcare：Web 服务到 DICOM”模式所使用的格式满足您的需求。
