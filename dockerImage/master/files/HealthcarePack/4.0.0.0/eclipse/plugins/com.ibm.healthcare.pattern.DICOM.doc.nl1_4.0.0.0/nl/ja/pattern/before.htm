「Healthcare: Web service to DICOM」パターンを適用する前に完了するタスク

パターン・インスタンスを作成するためにパターンを適用する前に、必要に応じて、以下の 1 つ以上のタスクを完了します。

  • 「Healthcare: Web service to DICOM」パターンがビジネス要件を満たしていること、および要求側アプリケーションと PACS の両方でサポート可能であることを確認します。
  • DICOM PACS アプリケーションを定義または識別し、それが C-FIND コマンドおよび C-MOVE コマンドをサポートしていることを確認します。
  • パターン・インスタンスに名前を付けます。適切な名前を選択できるように、パターン適用の目的を確認してください。
  • ロギング・メッセージやエラー・メッセージが必要かどうかを決定します。 どちらかのメッセージが必要な場合、どのキュー・マネージャーおよびキューにメッセージが送付されるかを決定する必要があります。複数のメッセージ・フローで使われる一般的なキューにメッセージを送信できます。または、パターン・インスタンス用の固有のキューを使用することもできます。
