Parameter für das Muster Web-Service an DICOM

DICOM- Service
Musterparameter Standardeinstellung Beschreibung
Titel der Anwendungsinstanz BROKERAE

Der DICOM-Titel der vom Muster generierten Anwendungsinstanz, wenn diese eine Verbindung zur fernen Anwendungsinstanz (PACS) herstellt.
Titel der fernen Anwendungsinstanz PACS

Der DICOM-Titel der fernen Anwendungsinstanz, an die die DICOM-Befehle C-FIND und C-MOVE gesendet werden.
Max. Anzahl Ergebnisse 99

Dieser Parameter gibt die maximale Anzahl an C-FIND-Antwortnachrichten an, die verarbeitet werden. Mit diesem Wert wird sichergestellt, dass C-FIND-Abfragen mit einer langen Ausführungszeit innerhalb eines angemessenen Zeitraums abgeschlossen werden.
Verbindungszeitlimit (Sekunden) 60

Der Zeitraum, der zum Herstellen einer Verbindung zur fernen Anwendungsinstanz (PACS) maximal zulässig ist.
Verbindungsdetails localhost:11112

Dieser Parameter definiert die TCP/IP-Verbindung für die ferne Anwendungsinstanz (PACS) im Format Hostname:Port.
Unterstützung für Abfrage-WSDL aktivieren Ausgewählt Bei Auswahl dieses Parameters gibt der Integrationsknoten beim Empfang einer HTTP-GET-Anforderung mit einer ?wsdl-Abfragezeichenfolge WSDL- und XML-Schemainformationen in Zusammenhang mit dem Endpunkt zurück. Über diesen Parameter kann die Verteilung der WSDL gesteuert werden.
Protokollierung
Musterparameter Standardeinstellung Beschreibung
Protokollierung erforderlich Nicht ausgewählt Dieser Musterparameter gibt an, ob die Musterinstanz einen untergeordneten Nachrichtenfluss für die Protokollierung enthält.

Wird dieser Parameter gelöscht, ist kein untergeordneter Nachrichtenfluss für die Protokollierung enthalten.

Bei Auswahl dieses Musterparameters müssen Sie für Protokollwarteschlangenmanager und Protokollwarteschlange einen Wert angeben.
Protokollwarteschlangenmanager Dieser Musterparameter gibt den Warteschlangenmanager für die Protokollierung an. Wenn der Warteschlangenmanager des Integrationsknotens für die Protokollfunktion verwendet wird, kann dieser Parameter leer bleiben.

Dieser Musterparameter ist nur erforderlich, wenn Protokollierung erforderlich ausgewählt ist.
Protokollwarteschlange LOG Dieser Musterparameter definiert die Warteschlange, die für die Protokollierung verwendet wird.

Dieser Musterparameter ist nur erforderlich, wenn Protokollierung erforderlich ausgewählt ist.
Fehlerbehandlung
Musterparameter Standardeinstellung Beschreibung
Fehlernachricht erforderlich Ausgewählt Dieser Musterparameter gibt an, ob die Musterinstanz Code zum Einreihen von Fehlernachrichten in eine Warteschlange enthält.

Bei Auswahl dieses Musterparameters müssen Sie für Fehlerwarteschlangenmanager und Fehlerwarteschlange einen Wert angeben.
Fehlerwarteschlangenmanager Dieser Musterparameter definiert den Warteschlangenmanager für die Fehlerprotokollierung.

Dieser Musterparameter ist nur erforderlich, wenn Fehlernachricht erforderlich ausgewählt ist.
Fehlerwarteschlange ERROR Dieser Musterparameter definiert die Warteschlange, die für Fehlernachrichten verwendet wird.

Dieser Musterparameter ist nur erforderlich, wenn Fehlernachricht erforderlich ausgewählt ist.
Allgemein
Musterparameter Standardeinstellung Beschreibung
Nachrichtenflusspräfix Dieser Musterparameter definiert das Präfix, das den Namen aller Nachrichtenflüsse vorangestellt wird.
Nachrichtenflusssuffix Dieser Musterparameter definiert das Suffix, das an den Namen aller Nachrichtenflüsse angehängt wird.
Warteschlangenpräfix Dieser Musterparameter definiert das Präfix, das allen Warteschlangennamen vorangestellt wird.

Bei Verwendung von Standardwarteschlangennamen muss das Präfix und/oder Suffix gesetzt werden, um Warteschlangennamen für die einzelnen Musterinstanzen anzugeben.
Warteschlangensuffix Dieser Musterparameter definiert das Suffix, das an alle Warteschlangennamen angehängt wird.

Bei Verwendung von Standardwarteschlangennamen muss das Präfix und/oder Suffix gesetzt werden, um Warteschlangennamen für die einzelnen Musterinstanzen anzugeben.
Kurzbeschreibung Dieser Musterparameter definiert die Beschreibung, die der Eigenschaft Kurzbeschreibung aller Nachrichtenflüsse hinzugefügt wird, die generiert werden.

Für alle Musterparameter sind die Standardwerte aufgeführt. Ist kein Standardwert angegeben, ist der Standardwert eine leere Zeichenfolge.
