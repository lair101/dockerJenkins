Tareas que deben completarse antes de aplicar el patrón Servicio sanitario: Servicio web a DICOM

Antes de aplicar el patrón para producir una instancia de patrón, complete una o más de las siguientes tareas, si es necesario:

  • Identifique que el patrón Servicio sanitario: Servicio web a DICOM satisface los requisitos de su empresa y puede estar soportado tanto por la aplicación solicitante como por el PACS.
  • Defina o identifique la aplicación PACS DICOM y asegúrese de que admite los mandatos C-FIND y C-MOVE.
  • Asigne un nombre a la instancia de patrón. Identifique la finalidad de la aplicación de patrón para poder elegir un nombre adecuado.
  • Decida si necesita mensajes de registro y de error. Si necesita cualquiera de los dos, debe decidir a qué gestor de colas y a qué cola se enviarán los mensajes. Puede enviar mensajes a una cola general que utilicen varios flujos de mensajes, o puede utilizar una cola específicamente para la instancia de patrón.
