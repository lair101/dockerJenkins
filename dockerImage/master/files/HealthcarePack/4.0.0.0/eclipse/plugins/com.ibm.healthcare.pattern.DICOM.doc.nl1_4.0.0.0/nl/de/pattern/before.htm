Tasks, die vor der Anwendung des Musters Healthcare: Web-Service an DICOM ausgeführt werden müssen

Vor der Anwendung einer Musterinstanz müssen Sie unter Umständen eine oder auch mehrere der folgenden Tasks ausführen:

  • Stellen Sie fest, ob das Muster 'Healthcare: Web-Service an DICOM' Ihre Geschäftsanforderungen erfüllt und sowohl von der anfordernden Anwendung als auch von PACS unterstützt wird.
  • Definieren bzw. ermitteln Sie die DICOM-PACS-Anwendung und stellen Sie sicher, dass sie die C-FIND- und C-MOVE-Befehle unterstützt.
  • Weisen Sie der Musterinstanz einen Namen zu. Wählen Sie dazu einen Namen aus, der den Zweck des Musters verdeutlicht.
  • Stellen Sie fest, ob eine Protokollierung und Fehlernachrichten erforderlich sind. Ist dies der Fall, müssen Sie den Warteschlangenmanager und die Warteschlange festlegen, an den bzw. die die Nachrichten gesendet werden sollen. Nachrichten können an eine allgemeine Warteschlange gesendet werden, die von mehreren Nachrichtenflüssen verwendet wird, oder an eine speziell für die Musterinstanz definierte Warteschlange.
