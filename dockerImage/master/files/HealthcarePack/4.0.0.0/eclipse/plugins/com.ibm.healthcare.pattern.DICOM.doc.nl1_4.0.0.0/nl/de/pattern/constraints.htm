Bedingungen für die Verwendung des Musters Healthcare: Web-Service an DICOM'

Für die Verwendung des Musters 'Healthcare: Web-Service an DICOM' gelten eine Reihe von Bedingungen.

Das Muster Healthcare: Web-Service an DICOM kann nur verwendet, werden wenn alle der folgenden Bedingungen erfüllt werden:

  • Anforderungen müssen als SOAP-Nachrichten über HTTP empfangen werden.
  • Anfordernde Anwendungen müssen einen synchronen Anfordung/Antwort-Interaktionsstil unterstützen.
  • Anfordernde Anwendungen müssen für vom Muster 'Healthcare: Web-Service an DICOM' definierte Suchkriterien und Ergebnisse das XML-Format unterstützen.
  • Das DICOM-PACS muss C-MOVE-Befehle unterstützen, wenn die DICOM-Instanzen in eine Zielanwendung verschoben werden.
  • Das DICOM-PACS muss für C-FIND-Befehle den Patienten- oder Untersuchungsabfragestamm unterstützen.
  • Wenn Protokollierung oder Fehlerbehandlung erforderlich ist, wird davon ausgegangen, dass das vom Muster 'Healthcare: Web-Service an DICOM' verwendete Format Ihre Anforderungen erfüllt.
