Parámetros para el patrón Servicio web a DICOM

Servicio DICOM
Parámetro de patrón Valor predeterminado Descripción
Título de entidad de aplicación BROKERAE

Es el título DICOM de la entidad de aplicación generada por patrón cuando se conecta a la entidad de aplicación remota (PACS).
Título de entidad de aplicación remota PACS

Es el título DICOM de la entidad de aplicación remota donde se envían los mandatos DICOM C-FIND y C-MOVE.
Número máximo de resultados 99

Este parámetro especifica el número máximo de mensajes de respuesta C-FIND que se procesan. Este valor garantiza C-FIND de consultas de larga ejecución completa en un periodo de tiempo razonable.
Tiempo de espera de conexión (segundos) 60

La cantidad máxima de tiempo permitido para establecer una conexión con la entidad de aplicación remota (PACS).
Información detallada de conexión localhost:11112

Este parámetro define la conexión TCP/IP para la entidad de aplicación remota (PACS) con el formato nombre_host:puerto.
Habilitar soporte para WSDL de consulta Seleccionado Si este parámetro está seleccionado, si el nodo de integración recibe una solicitud HTTP GET con una serie de consulta ?wsdl, el nodo de integración devuelve información WSDL y de esquema XML relacionada con el punto final. Utilice este parámetro para controlar la distribución de su WSDL.
Registro
Parámetro de patrón Valor predeterminado Descripción
Registro obligatorio Borrado Este parámetro de patrón determina si la instancia de patrón incluye el subflujo para el registro.

Si este parámetro de patrón no está seleccionado, no se incluye el subflujo de registro.

Si este parámetro de patrón está seleccionado, debe establecer valores para Gestor de colas de registro y Cola de registro.
Gestor de colas de registro Este parámetro de patrón define el gestor de colas que se utiliza para el registro. Puede dejar este parámetro en blanco si se utiliza el gestor de colas del nodo de integración para el registro.

Este parámetro de patrón sólo es necesario si se selecciona Registro obligatorio.
Cola de registro LOG Este parámetro de patrón define la cola que se utiliza para el registro.

Este parámetro de patrón sólo es necesario si se selecciona Registro obligatorio.
Manejo de errores
Parámetro de patrón Valor predeterminado Descripción
Mensaje de error obligatorio Seleccionado Este parámetro de patrón determina si la instancia de patrón incluye código para colocar mensajes de error en una cola.

Si este parámetro de patrón está seleccionado, se deben establecer valores para Gestor de colas de errores y Cola de errores.
Gestor de colas de errores Este parámetro de patrón define el gestor de colas que se utiliza para el registro de errores.

Este parámetro de patrón sólo es necesario si se selecciona Mensaje de error obligatorio.
Cola de errores ERROR Este parámetro de patrón define la cola que se utiliza para los mensajes de error.

Este parámetro de patrón sólo es necesario si se selecciona Mensaje de error obligatorio.
General
Parámetro de patrón Valor predeterminado Descripción
Prefijo de flujo Este parámetro de patrón define el prefijo que se añade al principio de todos los nombres de flujo de mensajes.
Sufijo de flujo Este parámetro de patrón define el sufijo que se añade al final de todos los nombres de flujos de mensajes.
Prefijo de cola Este parámetro de patrón define el prefijo que se añade al principio de todos los nombres de cola.

Si se utilizan nombres de cola predeterminados, se debe definir el prefijo, el sufijo, o ambos, para identificar los nombres de cola para cada instancia de patrón.
Sufijo de cola Este parámetro de patrón define el sufijo que se añade al final de todos los nombres de cola.

Si se utilizan nombres de cola predeterminados, se debe definir el prefijo, el sufijo, o ambos, para identificar los nombres de cola para cada instancia de patrón.
Descripción breve Este parámetro de patrón define la descripción que se añade a la propiedad Descripción corta de todos los flujos de mensajes que se generan.

Todos los parámetros de patrón aparecen en la lista con sus valores predeterminados. Cuando no se indica un valor predeterminado, éste es una serie vacía.
