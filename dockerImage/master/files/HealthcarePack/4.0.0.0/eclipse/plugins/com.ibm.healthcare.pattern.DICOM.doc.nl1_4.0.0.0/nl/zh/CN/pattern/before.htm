应用“Healthcare：Web 服务到 DICOM”模式前需要完成的任务

如果有需要，在应用此模式以生成模式实例之前，请完成下列一项或多项任务：

  • 确定“Healthcare：Web 服务到 DICOM”模式满足您的业务需求，并且同时受请求者应用程序和 PACS 支持。
  • 定义或确定 DICOM PACS 应用程序，并确保其支持 C-FIND 和 C-MOVE 命令。
  • 对模式实例进行命名。请确定模式应用程序的用途，以便选择合适的名称。
  • 确定是否需要日志记录和错误消息。如果需要其中任何一项，您必须确定要将消息发送到哪个队列管理器和队列。可以将消息发送到由多个消息流使用的常规队列，也可以使用特定于此模式实例的队列。
