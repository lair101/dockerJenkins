Healthcare

IBM Integration Bus Healthcare Pack builds on IBM Integration Bus to provide support for applications in healthcare.

Healthcare: HIPAA to XML pattern

The Healthcare: HIPAA to XML pattern generates a message flow that you can use to transform HIPAA files to XML files.

Healthcare: HL7 Transformation pattern

The Healthcare: HL7 Transformation pattern generates graphical data maps that you can use to assemble HL7 messages.

Healthcare: HL7 to HL7 DFDL pattern

The Healthcare: HL7 to HL7 DFDL pattern integrates an application that can send HL7 v2 messages with one or more applications that can receive HL7 messages. The applications must be capable of sending and receiving the messages by using MLLP over TCP/IP. This pattern uses the DFDL message model. An alternative version of this pattern, called the Healthcare: HL7 to HL7 pattern is provided if you want to use MRM and the HL7v25P message set instead of DFDL.

Healthcare: Medical devices to EMR pattern

The Healthcare: Medical devices to EMR pattern integrates medical devices with an Electronic Medical Record (EMR) application that can receive Health Level Seven International (HL7) v2 observation result messages (ORU R01). The application must be capable of receiving HL7 ORU R01 messages by using Minimal Lower Layer Protocol (MLLP) over TCP/IP.

Healthcare: Web service to DICOM pattern

Use the Healthcare: web service to DICOM pattern to integrate an application written using web services with DICOM applications that support C-FIND and C-MOVE operations. You can use the pattern to query patients, studies, series and images from a DICOM archive (PACS) using a web service implemented by IBM Integration Bus.

Healthcare: Home Health pattern

Use the Healthcare: Home Health pattern to enable your health systems to accept web service requests from HL7 v2.6-compliant devices. This allows data that Home Health devices collect and send to the Application Hosting Device (AHD) to then be sent to the requesting clinical application.

Healthcare: Patient Identifier Cross-reference Manager pattern

Use the Healthcare: Patient Identifier Cross-reference Manager pattern to create a Patient Identifier Cross-reference Manager that you can use with the IHE PIX profile, to add patients to an EMPI and retrieve patient IDs.

Healthcare: Patient Demographics Query Supplier pattern

Use the Healthcare: Patient Demographics Query Supplier pattern to create a Patient Demographics Supplier that you can use with the IHE PDQ profile, to retrieve patient demographic information.

Healthcare: Cross-Enterprise Document Sharing Consumer pattern

Use the Healthcare: Cross-Enterprise Document Sharing Consumer pattern to create an integration solution that you can use with the IHE XDS profile, to query a document registry and retrieve documents from a document repository.

Healthcare: FHIR Transformation pattern

Use the Healthcare: FHIR Transformation pattern to create an application that you can use to transform HL7 FHIR resources between XML and JSON formats.
