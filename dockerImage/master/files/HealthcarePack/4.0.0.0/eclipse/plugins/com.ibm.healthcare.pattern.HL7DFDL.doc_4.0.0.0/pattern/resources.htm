Resources for the Healthcare: HL7 to HL7 DFDL pattern

Note: You can use the links in this page only when you access the pattern from the Patterns Explorer in the IBM Integration Toolkit.

The Healthcare: HL7 to HL7 DFDL pattern uses a DFDL library to parse, validate and transform HL7 messages.

You can import the HL7 DFDL libraries by clicking the following links:

  • Import the HL7 v251 DFDL models into your workspace
  • Import the HL7 v26 DFDL models into your workspace
  • Import the HL7 v27 DFDL models into your workspace

You can import the HL7 test messages and ATNA schemas by clicking the following link:

  • Import the resources into your workspace

The following resources are not required for running instances of the Healthcare: HL7 to HL7 DFDL pattern. If you want to debug the HL7DFDLInput or HL7DFDLOutput nodes, you can import the source code for the nodes by clicking the following link:

  • Import the source code resources into your workspace

When you finish using the source code resources, delete them from your workspace.
