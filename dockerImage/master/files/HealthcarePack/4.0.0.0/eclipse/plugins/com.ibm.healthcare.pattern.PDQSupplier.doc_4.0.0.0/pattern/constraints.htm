Constraints on the use of the Healthcare: Patient Demographics Query Supplier pattern

To use the Healthcare: Patient Demographics Query Supplier pattern, you must meet the following constraints:

  • You must use a supported version of IBM® InfoSphere® Master Data Management (MDM) as your EMPI. For more information about supported versions, in the IBM Integration Bus Healthcare Pack documentation, go to "Platform support for IBM Integration Bus Healthcare Pack" and click on "IBM Integration Bus system requirements."
  • In IBM InfoSphere MDM, your Master Data Management Project must use the Patient template.
