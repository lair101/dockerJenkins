Tasks to complete before applying the Patient Demographics Query Supplier pattern

Before you use the pattern and generate a pattern instance, complete one or more of the following tasks, if required:

  • Identify that the Patient Demographics Query Supplier pattern meets your business requirement.
  • Install and configure IBM® InfoSphere® Master Data Management (MDM) as your EMPI. Ensure the following items are configured:
    • Your Master Data Management Project must use the Patient template.
  • Identify the URL of the web service that is used by your EMPI and the username and password that are required to connect to the EMPI.
