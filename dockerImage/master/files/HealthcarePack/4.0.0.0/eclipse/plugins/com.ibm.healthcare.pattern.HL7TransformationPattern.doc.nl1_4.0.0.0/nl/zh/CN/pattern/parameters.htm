“HL7 变换”模式的参数

常规配置选项
模式参数 缺省值 描述
从以下位置派生出库 HL7 消息

此参数指定是想要基于 HL7 消息定义还是基于单个 HL7 段定义（然后可进行组合以生成自己的变换）生成库内容。
分配值

此参数指定要针对 HL7 消息中的字段指定的字符串值。
生成 图形数据图

此参数控制生成的库是否应包含“图形数据图”或 ESQL 代码。
逐个构造 MSH 字段 选中

如果想要通过为每个字段指定不同的值来生成 MSH 段变换，那么请选中此选项。如果不选中此选项，那么 MSH 段将视为与其他段相同，并且将为每个字段指定相同的值。
创建示例消息流 选中

如果想要生成一个示例消息流并且该消息流使用生成的变换逻辑，请选中此选项。如果要从 HL7 段衍生此库，那么将忽略此选项。
