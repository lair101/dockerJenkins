「HL7 Transformation」パターンのパラメーター

一般構成オプション
パターン・パラメーター デフォルト 説明
ライブラリーの派生元 HL7 メッセージ

このパラメーターは、HL7 メッセージ定義または個別の HL7 セグメント定義 (後から結合して独自の変換を生成する) に基づいてライブラリーを生成するかを指定します。
割り当て値

このパラメーターは、HL7 メッセージ内のフィールドに割り当てられるストリング値を指定します。
生成 Gグラフィカル・データ・マップ

このパラメーターでは、生成するライブラリーにグラフィカル・データ・マップを入れるか、ESQL コードを入れるかを制御します。
MSH フィールドを個別に構成 選択

フィールドごとに別々の値を割り当てて MSH セグメント変換を生成するときに、このオプションを選択します。このオプションを選択しない場合は、MSH セグメントが他のセグメントと同じように処理され、各フィールドに同じ値が割り当てられます。
サンプル・メッセージ・フローの作成 (Create an example Message Flow) 選択

生成する変換ロジックを使用したサンプル・メッセージ・フローを生成するときに、このオプションを選択します。HL7 セグメントからライブラリーを派生させる場合、このオプションは無視されます。
