Parámetros para el patrón Transformación de HL7

Opciones generales de configuración
Parámetro de patrón Valor predeterminado Descripción
Derivar biblioteca de Mensajes HL7

Este parámetro especifica si desea generar el contenido de la biblioteca en base a las definiciones de mensajes HL7, o en base a definiciones individuales de segmentos HL7 (que luego puede combinar para generar sus propias transformaciones).
Valor de asignación

Este parámetro especifica un valor de serie que se asignará a los campos dentro de los mensajes HL7.
Generar Correlación de datos gráficos

Este parámetro controla si la biblioteca generada debe contener correlaciones de datos gráficos o código ESQL.
Construir campos MSH individualmente Seleccionado

Seleccione esta opción si desea generar una transformación de segmento MSH con distintos valores asignados para cada campo. Si esta opción no está seleccionada, el segmento MSH se tratará igual que otros segmentos y los valores asignados para cada campo serán el mismo.
Crear un flujo de mensajes de ejemplo Seleccionado

Seleccione esta opción si desea generar un flujo de mensajes de ejemplo que utilice la lógica de transformación generada. Si va a derivar la biblioteca de Segmentos HL7, esta opción se ignorará.
