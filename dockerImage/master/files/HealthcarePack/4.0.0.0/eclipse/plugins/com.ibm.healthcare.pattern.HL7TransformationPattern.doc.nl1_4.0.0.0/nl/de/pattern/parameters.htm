Parameter für das Muster HL7-Transformation

Allgemeine Konfigurationsoptionen
Musterparameter Standardeinstellung Beschreibung
Bibliothek ableiten von HL7-Nachrichten

Dieser Parameter gibt an, ob Sie den Bibliotheksinhalt basierend auf HL7-Nachrichtendefinitionen oder basierend auf einzelnen HL7-Segmentdefinitionen generieren möchten (die Sie kombinieren können, um eigene Transformationen zu generieren).
Zuordnungswert

Dieser Parameter gibt einen Zeichenfolgewert an, der Feldern in den HL7-Nachrichten zugeordnet werden soll.
Generieren Grafische Datenzuordnung

Dieser Parameter steuert, ob die generierte Bibliothek grafische Datenzuordnungen oder ESQL-Code enthalten soll.
MSH-Felder einzeln erstellen Ausgewählt

Aktivieren Sie diese Option, wenn eine MSH-Segmenttransformation mit unterschiedlichen zugeordneten Werten für die einzelnen Felder generiert werden soll. Wenn diese Option nicht aktiviert wird, wird das MSH-Segment genau wie andere Segmente behandelt und die zugeordneten Werte der einzelnen Felder bleiben identisch.
Beispielnachrichtenfluss erstellen Ausgewählt

Aktivieren Sie diese Option, wenn ein Beispielnachrichtenfluss generiert werden soll, der die generierte Transformationslogik verwendet. Falls Sie Ihre Bibliothek aus HL7-Segmenten ableiten, wird diese Option ignoriert.
