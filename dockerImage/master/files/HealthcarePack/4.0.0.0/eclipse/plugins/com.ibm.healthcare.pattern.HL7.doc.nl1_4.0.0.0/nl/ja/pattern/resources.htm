「Healthcare: HL7 to HL7」パターンのリソース

注: このページ内のリンクは、IBM Integration Toolkit のパターン・エクスプローラーからパターンにアクセスするときのみ使用できます。

「Healthcare: HL7 to HL7」パターンはメッセージ・セットを使用して、HL7 メッセージの構文解析、検証、および変換を行います。

以下のリンクをクリックすることにより、HL7 メッセージ・セット、ATNA スキーマ、およびテスト・リソースをインポートできます。

  • サンプル・リソースをワークスペースにインポートします。

サンプル・リソースを使用し終わったら、それらをワークスペースから削除します。

以下のリンクをクリックすることにより、GenericHL7Input ノードおよび GenericHL7Output ノードのソース・コードをインポートできます。

  • ソース・コード・リソースをワークスペースにインポートします。

ソース・コード・リソースの使用を終えたら、それらをワークスペースから削除します。
