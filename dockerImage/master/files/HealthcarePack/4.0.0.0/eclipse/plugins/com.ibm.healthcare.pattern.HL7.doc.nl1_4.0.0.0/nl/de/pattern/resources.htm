Ressourcen für das Muster 'Healthcare: HL7 an HL7'

Hinweis: Die Links auf dieser Seite können nur verwendet werden, wenn Sie über den Musterexplorer im IBM Integration Toolkit auf das Muster zugreifen.

Das Muster Healthcare: HL7 an HL7 verwendet eine Nachrichtengruppe für Parsing, Validierung und Transformation von HL7-Nachrichten.

Über den folgenden Link können Sie die HL7-Nachrichtengruppe, die ATNA-Schemas und die Testressourcen importieren:

  • Beispielressourcen in Ihren Arbeitsbereich importieren

Wenn Sie mit der Verarbeitung der Beispielressourcen fertig sind, löschen Sie diese aus Ihrem Arbeitsbereich.

Sie können den Quellcode für den GenericHL7Input- und den GenericHL7Output-Knoten importieren, indem Sie auf den folgenden Link klicken:

  • Quellcoderessourcen in Ihren Arbeitsbereich importieren

Wenn Sie die Arbeit mit den Quellcode-Ressourcen beendet haben, löschen Sie sie aus Ihrem Arbeitsbereich.
