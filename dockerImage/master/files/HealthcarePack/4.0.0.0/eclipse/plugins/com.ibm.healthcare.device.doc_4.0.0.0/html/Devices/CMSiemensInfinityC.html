Computer connection

The driver supports the RS232 connection.

Please read the following description carefully and ensure the cable in use matches the monitor.

- For Delta, Kappa, Kappa XLT, SC7000, SC8000, SC9000 the cable connections must be developed from Drager cable; part number 5206441. This is a universally keyed interface cable with 14-pin Minichamp connector at one end and un-terminated on the other.
- The Kappa XLT must be developed from Drager cable; part number MS15045.

Serial connection location on the Infinity device (depending on configuration):

- Delta, Delta XL, SC7000, SC9000XL connects to X3 or X5 IDS or CPS connector.
- The Kappa or SC8000 connects to X5 only.
- The Kappa XLT connects to Com3 port

Notes:
The use of an IDS / CPS is required to simultaneously support a SC 9015 and data export. The X5 port on the IDS / CPS is used for the SC 9015 and the X3 port for RS 232 data interface.
A DirectNet cannot support a SC 9015 and data interfacing protocol at the same time.
The RS 232 interfacing is not available when used with a Sirenet CPS

Serial Connections to X3 CPS / IDS connector:
The unterminated end should be wired as follows:

Patient Monitor
PC
Signal 14 Pin Minichamp Signal 9 pin ‘D’ 25 pin ‘D’
GND 10(grey) GND 5 7
TxD 12 (Pink) RxD 3 2
RxD 13 (Lt Blue) TxD 2 3

 


Serial Connections to X5 or X5 IDS or CPS connector:
The unterminated end should be wired as follows:

Patient Monitor
PC
Signal 14 Pin Minichamp Signal 9 pin ‘D’ 25 pin ‘D’
GND 7 (Blue) GND 5 7
TxD 10 (Grey) RxD 2 3
RxD 11 (Violet) TxD 3 2
