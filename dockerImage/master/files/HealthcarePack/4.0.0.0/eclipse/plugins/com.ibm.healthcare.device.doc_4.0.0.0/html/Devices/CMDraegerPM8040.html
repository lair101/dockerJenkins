Computer Connection

The driver supports the RS232 connection.

Please read the following description carefully and ensure the cable in use matches the monitor.

Connectors on Port 1

Computer connection location: rear side of PM8040 Label: Rec.

Type RS-232-C 15 pin Sub D (female)

Housing Shield on connector's housing
RxD 10
TxD 11
GND 12
Cable 1.5kV galvanic isolation

Connectors on Port 2 (default port)

Computer connection location: rear side of PM8040 Label: Com.

The cable is a straight-through cable and should be wired as follows:

Type RS-232-C 9 pin Sub D (female)

Housing Shield on connector's housing
RxD 6
TxD 7
GND 4
Cable 1.5kV galvanic isolation

Supported communication parameters:

Port Configuration (both ports independent):

Baud rate 1200
Parity E
Data bits 8
Start bits 1
Stop bits 1

Port 2 is the default port where MEDIBUS is installed by factory. MEDIBUS can be installed only on one port at a time.

When MEDIBUS is installed on either port it is automatically uninstalled from the other port.
