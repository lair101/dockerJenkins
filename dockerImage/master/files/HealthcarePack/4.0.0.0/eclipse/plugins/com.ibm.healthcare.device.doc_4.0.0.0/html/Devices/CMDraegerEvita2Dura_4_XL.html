Computer Connection

The driver supports the RS232 connection.

Please read the following description carefully and ensure the cable in use matches the monitor.

Computer connection location: Rear of Evita 4 / Evita 2 dura / Evita XL Label: COM1

or on optional extension board "Evita 4 Link", Label COM2 or COM3

The cable is a straight-through cable and should be wired as follows:

Type RS-232-C 9-pin Sub D (female)

Housing 1
RxD 2
TxD 3
GND 5
Cable 1.5kV galvanic isolation

Supported communication parameters:

Port Configuration Channel B

Baud rate 1200, 2400, 4800, 9600,19200
Parity E,O,N
Data bits 8
Start bits 1
Stop bits 1,2
