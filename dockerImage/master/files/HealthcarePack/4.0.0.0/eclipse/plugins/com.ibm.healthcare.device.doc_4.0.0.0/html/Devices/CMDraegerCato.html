Computer Connection

The driver supports the RS232 connection.

Please read the following description carefully and ensure the cable in use matches the monitor.

Computer connection location: rear side of Cato Labels Port 1: RS-232-C Port 2: Protokoll/ Printer/ RS-232-C

Type RS-232-C 25 pin Sub D (female)

Housing 1
RxD 3
TxD 2
GND 7
Cable 1.5kV galvanic isolation

Supported communication parameters:

Port Configuration (both ports independent):

Baud rate 1200, 9600
Parity E
Data bits 8
Start bits 1
Stop bits 1

Port 2 must be configured to run MEDIBUS instead of Printer-protocol

Port 2 MEDIBUS is provided for Cato device version 2.00 and higher.
