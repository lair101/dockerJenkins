Computer connection

The driver supports the RS232 connection.

Please read the following description carefully and ensure the cable in use matches the monitor.

Cable connections must be developed from Draeger cable; part number 5206441. This is a universally keyed interface cable with 14-pin Minichamp connector at one end and un-terminated on the other.

Serial connection location: The 14-pin Minichamp connector end of the cable connects to the X5 connector on the Infinity CPS or IDS or on the Interface plate (p/n 3376493). Note: To support the Scio mutli gas; connect Scio mutli gas module to the USB port.

The unterminated end should be wired as follows:

Patient Monitor
PC
Signal 14 Pin Minichamp Signal 9 pin ‘D’ 25 pin ‘D’
GND 7 (Blue) GND 5 7
TxD 10 (Grey) RxD 2 3
RxD 11 (Violet) TxD 3 2

 


Supported communication parameters:

Fixed baud rate must be used

Baud rate 19200
Parity None
Data bits 8
Stop bits 1
Flow control None
