Computer Connection

The driver supports the RS232 connection.

Please read the following description carefully and ensure the cable in use matches the monitor.

Computer connection location: rear side of PM8060 Vitara Labels Port 1: 'RS 232 C' Port 2 'RS 232 C' + Protocol Symbol

The cable is a crossed cable and should be wired as follows:

Type RS-232-C 9 pin Sub D (female)

Housing Shield on connector's housing
RxD 3
TxD 2
GND 5
Cable 500 V galvanic isolation

Supported communication parameters:

Port Configuration (both ports independent):

Baud rate 1200, 9600
Parity E
Data bits 8
Start bits 1
Stop bits 1
