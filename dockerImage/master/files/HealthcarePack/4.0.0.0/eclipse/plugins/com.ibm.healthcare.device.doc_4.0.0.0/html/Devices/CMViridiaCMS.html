Supported devices: Philips CMS, Viridia 24, Viridia 26
Release 1.10.0.0
Document reference: 1a

Known issues:

  • CMS monitor gives temperature in daC (degrees C x 10).
  • Bandwidth limitations means the monitor can not transmit all possible data parameters in a single request. Please limit requests to only the required data.

Computer connection

The serial port of the device is at the back, and is the lower 25 pin (DB25) port marked RS232.

The device requires a serial cable with the following pin layout:

For communication settings on the device, please refer to a Philips support engineer, or the Philips document M1046-9420L (Philips CMS Patient Monitoring System & V24 and V26 Patient Monitors RS232 Computer Interface Programming Guide).

Supported communication parameters:

Baud rate 9600, 19200 (default), 38400
Parity None
Data bits 8
Stop bits 1
Flow control RTS/CTS
