Computer connection

The driver supports both LAN and serial (RS232) connections.

The serial port of the device is at the back, and is the upper RJ45 port.

The LAN port of the device is at the back and is the lower RJ45 port marked LAN. Each monitor is assigned an IP address and a monitor identifier (either by the monitor user or through a DHCP server). One or both must be used in the driver configuration in order to establish a connection.

The device requires a serial cable with the following pin layout (The pins of the RJ45 are counted from 1 for the lowest pin to 8 for the highest pin when looking at
the RS232/MIB interface board):
