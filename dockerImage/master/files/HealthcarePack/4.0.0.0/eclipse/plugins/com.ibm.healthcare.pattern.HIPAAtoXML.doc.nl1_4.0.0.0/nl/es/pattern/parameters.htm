Parámetros para el patrón HIPAA a XML

General
Parámetro de patrón Valor predeterminado Descripción
Directorio de entrada

Este parámetro especifica el directorio que contiene los archivos HIPAA de entrada.
Nombre de archivo o patrón de entrada

Este parámetro especifica el nombre de archivo del archivo de entrada HIPAA que se procesa. Puede utilizar caracteres comodín para especificar más de un archivo.
Directorio de salida

Este parámetro especifica el directorio donde se guardan los archivos XML de salida.
Directorio para anomalías

Este parámetro especifica el directorio donde se guarda el archivo de entrada si se produce algún error de análisis o de correlación.

Todos los parámetros de patrón aparecen en la lista con sus valores predeterminados. Cuando no se indica un valor predeterminado, éste es una serie vacía.
