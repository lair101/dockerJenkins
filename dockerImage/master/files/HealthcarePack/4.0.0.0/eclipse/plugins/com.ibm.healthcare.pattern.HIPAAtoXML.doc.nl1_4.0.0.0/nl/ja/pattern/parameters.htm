「HIPAA から XML へ」パターンのパラメーター

その他
パターン・パラメーター デフォルト 説明
入力ディレクトリー

このパラメーターは、HIPAA 入力ファイルを含むディレクトリーを指定します。
入力ファイル名またはパターン

このパラメーターは、処理される HIPAA 入力ファイルのファイル名を指定します。ワイルドカードを使用すると、複数のファイルを指定できます。
出力ディレクトリー

このパラメーターは、出力 XML ファイルの書き込み先のディレクトリーを指定します。
障害用ディレクトリー

このパラメーターは、構文解析やマッピングに障害が生じた場合に入力ファイルの書き込み先となるディレクトリーを指定します。

すべてのパターン・パラメーターとそのデフォルト値をリストします。 デフォルトが示されていない場合、デフォルト値は空ストリングです。
