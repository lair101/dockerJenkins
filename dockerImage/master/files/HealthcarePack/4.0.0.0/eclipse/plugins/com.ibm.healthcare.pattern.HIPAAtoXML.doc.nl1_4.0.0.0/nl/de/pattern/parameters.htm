Parameter für das Muster HIPAA in XML

Allgemein
Musterparameter Standard Beschreibung
Eingabeverzeichnis

Dieser Parameter gibt das Verzeichnis an, das die HIPAA-Eingabedateien enthält.
Eingabedateiname oder Muster

Dieser Parameter gibt den Dateinamen der HIPAA-Eingabedatei an, die verarbeitet wird. Sie können Platzhalter verwenden, um mehrere Dateien anzugeben.
Ausgabeverzeichnis

Dieser Parameter gibt das Verzeichnis an, in das die XML-Ausgabedateien geschrieben werden.
Verzeichnis für Fehler

Dieser Parameter gibt das Verzeichnis an, in das die Eingabedatei geschrieben wird, wenn ein Parsing- oder Zuordnungsfehler auftritt.

Für alle Musterparameter sind die Standardwerte aufgeführt. Ist kein Standardwert angegeben, ist der Standardwert eine leere Zeichenfolge.
