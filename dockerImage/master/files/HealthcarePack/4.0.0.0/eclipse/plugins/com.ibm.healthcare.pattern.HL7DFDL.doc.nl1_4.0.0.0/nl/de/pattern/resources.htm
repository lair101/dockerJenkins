Ressourcen für das Muster 'Healthcare: HL7 an HL7 DFDL'

Hinweis: Die Links auf dieser Seite können nur verwendet werden, wenn Sie über den Musterexplorer im IBM Integration Toolkit auf das Muster zugreifen.

Das Muster Healthcare: HL7 an HL7 DFDL analysiert, überprüft und transformiert HL7-Nachrichten mithilfe einer DFDL-Bibliothek.

Sie können die HL7-DFDL-Bibliotheken über die folgenden Links importieren:

  • HL7-DFDL-Modelle der Version 2.5.1 in den Arbeitsbereich importieren
  • HL7-DFDL-Modelle der Version 2.6 in den Arbeitsbereich importieren
  • HL7-Modelle der Version 2.7 in den Arbeitsbereich importieren

Sie können die Testressourcen und ATNA-Schemas über den folgenden Link importieren:

  • Beispielressourcen in den Arbeitsbereich importieren

Sie können den Quellcode der HL7DFDLInput- und HL7DFDLOutput-Knoten über den folgenden Link importieren:

  • Quellcoderessourcen in den Arbeitsbereich importieren

Wenn Sie die Quellcoderessourcen nicht mehr benötigen, können Sie sie aus dem Arbeitsbereich löschen.
