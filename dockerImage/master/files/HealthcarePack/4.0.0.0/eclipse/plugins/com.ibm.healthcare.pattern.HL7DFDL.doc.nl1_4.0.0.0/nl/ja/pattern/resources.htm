「Healthcare: HL7 to HL7 DFDL」パターンのリソース

注: このページ内のリンクは、IBM Integration Toolkit のパターン・エクスプローラーからパターンにアクセスするときのみ使用できます。

「Healthcare: HL7 to HL7 DFDL」パターンは、DFDL ライブラリーを使用して HL7 メッセージを解析、検証、および変換します。

以下のリンクをクリックすると、HL7 DFDL ライブラリーをインポートできます。

  • HL7 v251 DFDL モデルをワークスペースにインポートします
  • HL7 v26 DFDL モデルをワークスペースにインポートします
  • HL7 v27 DFDL モデルをワークスペースにインポートします

以下のリンクをクリックすると、テスト・リソースと ATNA スキーマをインポートできます。

  • サンプル・リソースをワークスペースにインポートします。

以下のリンクをクリックすると、HL7DFDLInput および HL7DFDLOutput ノードのソース・コードをインポートできます。

  • ソース・コード・リソースをワークスペースにインポートします。

サンプル・リソースを使用し終わったら、それらをワークスペースから削除します。
