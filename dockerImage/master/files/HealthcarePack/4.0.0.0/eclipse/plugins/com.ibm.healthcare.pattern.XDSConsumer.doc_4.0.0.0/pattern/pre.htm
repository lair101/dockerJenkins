Tasks to complete before applying the Healthcare: Cross-Enterprise Document Sharing Consumer pattern

If required, complete one or both of the following tasks before using the Healthcare: Cross-Enterprise Document Sharing Consumer pattern.

Secure endpoint connection to your document registry and document repository

Use SSL to access your document registry and document repository by associating the SSL certificates with the integration node either at the integration node level or at the integration server level, and then specify the HTTPS address as your endpoint. For more information, see Setting up a public key infrastructure.

Use shared libraries or a shared class loader to handle the pattern .jar file dependencies

Use a shared class loader or shared library to speed up deployment times. If you use a shared class loader or shared library, you do not need to deploy all of the supplied .jar files when you deploy your BAR file. For more information, see JavaCompute node class loading and JavaCompute node class loading by using a configurable service.
