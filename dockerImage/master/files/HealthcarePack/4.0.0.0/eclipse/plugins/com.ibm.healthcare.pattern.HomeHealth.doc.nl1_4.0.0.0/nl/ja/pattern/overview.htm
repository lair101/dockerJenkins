「Healthcare: Home Health」パターン

「Healthcare: Home Health」パターンを使用して、ホーム・ヘルス・デバイスのデータを送信するモバイル・デバイスまたは PC デバイスからの Web サービス要求を受け付けます。

コンテキスト

患者は、体重計や心臓モニターなどのホーム・ヘルス・デバイスを使用して自分の健康状態を確認しています。これらのデバイスをアプリケーション・ホスティング・デバイス (モバイル・デバイスや家庭用 PC など) に接続し、その情報を転送します。医療従事者は、AHD の読み取りデータを調べて、患者の健康状態をモニターしなければなりません。患者の健康状態を正確にモニターするには、定期的な読み取りが必要です。

目標

医療従事者は、広域ネットワーク (WAN) 経由でホーム・ヘルス・デバイスのデータを受け取り、要求元のアプリケーションでそのデータを表示します。そのためには、ホーム・ヘルス・デバイスの情報を受け取る AHD と要求元のアプリケーションを統合したシステムが必要です。そのシステムでは、HL7 メッセージ・タイプ Unsolicited Observation Results (ORU_R01) のメッセージを受け取り、メッセージの受け取りに関する確認応答を返す機能が必要になります。

ソリューション

「Healthcare: Home Health」パターンは、以下の機能を備えた開発アクセラレーターです。

  • 受信した SOAP 要求から HL7 メッセージ・タイプ Unsolicited Observation Results (ORU_R01) のメッセージを抽出して確認するフローとサブフロー。
  • 受信したメッセージに対してユーザー定義の処理を実行する機能。
  • 適切な確認応答をクライアントに送り返す機能。
  • SAML を使用してセキュリティーをセットアップするための構成可能サービス・スクリプト。
