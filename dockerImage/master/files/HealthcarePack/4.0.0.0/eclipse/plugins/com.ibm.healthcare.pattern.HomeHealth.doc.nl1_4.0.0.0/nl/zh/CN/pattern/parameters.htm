“Healthcare：Home Health”模式的参数

SOAP_Input
模式参数 缺省值 描述
URL 的路径后缀 /DeviceObservationConsumer

此属性指示从中检索 Web 服务请求的位置。请勿使用完整的 URL。如果想要的 URL 为 http://hostname[:port]/[path]，那么指定 /path 或 /path fragment/*，其中 * 是可用于表示匹配任何值的通配符。
安全性
模式参数 缺省值 描述
包含 SAML 2.0 令牌通过 选中

选择此框将包含 PEP 节点以启用 SAML 2.0 令牌的外部授权和认证

所有模式参数均随其缺省值一起列出。 未指示缺省值时，表明缺省值是空字符串。
