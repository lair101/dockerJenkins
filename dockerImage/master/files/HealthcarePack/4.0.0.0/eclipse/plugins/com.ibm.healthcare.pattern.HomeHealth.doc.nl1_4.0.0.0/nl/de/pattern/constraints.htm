Bedingungen für das Muster 'Healthcare: Ambulante Pflege'

Damit Sie das Muster 'Healthcare: Ambulante Pflege' verwenden können, müssen Sie die folgenden Bedingungen erfüllen:

  • Stellen Sie sicher, dass Ihre Geräte für die ambulante Pflege und Anwendungshostgeräte den HL7 v2.6-Nachrichtentyp Unsolicited Observation Results (Nicht angeforderte Beobachtungsergebnisse) verwenden und die Ergebnisse unter Verwendung von SOAP senden können.
