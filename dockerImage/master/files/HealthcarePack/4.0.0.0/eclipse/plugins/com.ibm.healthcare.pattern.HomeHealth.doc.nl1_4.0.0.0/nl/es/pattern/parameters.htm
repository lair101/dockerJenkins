Parámetros de patrón para el patrón Servicio sanitario: Salud en el hogar

SOAP_Input
Parámetro de patrón Valor predeterminado Descripción
Sufijo de vía de acceso del URL /DeviceObservationConsumer

Esta propiedad identifica la ubicación desde la que se recuperan las solicitudes de servicio web. No utilice el URL completo. Si el URL que desea es http://hostname[:puerto]/[vía_acceso], especifique /vía_acceso o bien /vía_acceso fragmento/*, donde * es un comodín que puede utilizar para indicar coincidir con cualquiera.

Seguridad
Parámetro de patrón Valor predeterminado Descripción
Incluir paso a través de señal SAML 2.0 Seleccionado

Si se selecciona este recuadro, se incluirá un nodo PEP para habilitar la autorización y autenticación externas de una señal SAML 2.0

Todos los parámetros de patrón aparecen en la lista con sus valores predeterminados. Cuando no se indica un valor predeterminado, éste es una serie vacía.
