Restricciones para el patrón Servicio sanitario: Salud en el hogar

Para utilizar el patrón Servicio sanitario: Salud en el hogar, debe cumplir las siguientes restricciones:

  • Asegúrese de que los dispositivos de Salud en el hogar y los AHD utilizan el tipo de enviar un mensaje HL7 v2.6 Unsolicited Observation Results, y pueden enviarlos utilizando SOAP.
