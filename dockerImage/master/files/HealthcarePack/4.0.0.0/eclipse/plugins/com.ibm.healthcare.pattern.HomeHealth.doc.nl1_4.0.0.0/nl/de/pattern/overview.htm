Muster 'Healthcare: Ambulante Pflege'

Verwenden Sie das Muster 'Healthcare: Ambulante Pflege' zum Akzeptieren von Web-Service-Anforderungen von mobilen Geräten oder PC-Geräten, die Daten aus den Geräte für die ambulante Pflege senden.

Kontext

Patienten besitzen Geräte, zum Beispiel Waagen oder Herzüberwachungsgeräte, die für die häusliche Überwachung des Gesundheitszustands vorgesehen sind. Mit diesen Geräten können die Patienten ihren Gesundheitszustand anhand von Messwerten überprüfen. Diese Geräte verbinden sich mit einem Anwendungshostgerät wie einem mobilen Gerät oder Home-PC, das diese Informationen weiterleiten kann. Das medizinische Fachpersonal muss die gemessenen Werte aus den Anwendungshostgeräten kennen, damit es den Gesundheitszustand seiner Patienten überwachen kann. Für eine präzise Überwachung des Gesundheitszustands der Patienten müssen die Werte regelmäßig abgelesen werden.

Ziel

Eine medizinische Fachkraft möchte Daten aus den Geräten für die ambulante Pflege über ein Weitverkehrsnetz (WAN) empfangen und die Daten in ihrer anfordernden Anwendung anzeigen. Sie benötigt ein System, das ihre anfordernde Anwendung in ein Anwendungshostgerät integriert, welches Informationen vom Gerät für die ambulante Pflege empfängt. Dieses System muss in der Lage sein, Nachrichten des HL7-Nachrichtentyps Unsolicited Observation Results (ORU_R01) (Nicht angeforderte Beobachtungsergebnisse) zu empfangen. Außerdem muss es Empfangsbestätigungen bezüglich der Nachrichten zurücksenden können.

Lösung

Das Muster 'Healthcare: Ambulante Pflege' beschleunigt die Entwicklung und stellt folgende Funktionen bereit:

  • Flüsse und untergeordnete Flüsse, die den HL7-Nachrichtentyp Unsolicited Observation Results (ORU_R01) (Nicht angeforderte Beobachtungsergebnisse) von empfangenen SOAP-Anforderungen extrahieren und prüfen
  • Benutzerdefinierte Verarbeitung der empfangenen Nachricht
  • Rücksenden der entsprechenden Bestätigung an den Client
  • Konfigurierbare Service-Scripts, mit denen Sie unter Verwendung der Security Assertion Markup Language (SAML) die Sicherheit einrichten können
