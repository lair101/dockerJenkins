Pfadsuffix für URL /DeviceObservationConsumer

Diese Eigenschaft gibt die Position an, von der aus Web-Service-Anforderungen abgerufen werden. Verwenden Sie nicht die vollständige URL. Wenn die gewünschte URL http://Hostname[:Port]/[Pfad] lautet, geben Sie entweder '/path' (Pfad) oder '/path fragment/*' (Pfadfragment) an. Dabei steht das * als Platzhalter für beliebige Übereinstimmungen.

Wenn Sie dieses Kästchen aktivieren, wird ein PEP-Knoten eingeschlossen, damit die externe Autorisierung und Authentifizierung eines SAML 2.0-Tokens möglich ist.

Für alle Musterparameter sind die Standardwerte aufgeführt. Ist kein Standardwert angegeben, ist der Standardwert eine leere Zeichenfolge.
