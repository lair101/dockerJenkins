Tareas que deben realizarse antes de aplicar el patrón Patient Identifier Cross-reference Manager

Antes de utilizar el patrón y generar una instancia de patrón, complete una o más de las siguientes tareas, si es necesario:

  • Identifique que el patrónPatient Identifier Cross-reference Manager cumple los requisitos de su empresa.
  • Instale y configure IBM® InfoSphere® Master Data Management (MDM) como su EMPI. Asegúrese de que los elementos siguientes están configurados:
    • Su Proyecto de gestión de datos maestros debe utilizar la plantilla Paciente.
    • El Dominio de identificación de pacientes de su EMPI debe estar configurado y debe definir el correspondiente srcCode.
  • Identifique el URL del servicio web que utiliza su EMPI y el nombre de usuario y contraseña que se necesitan para establecer conexión con el EMPI.
