“Healthcare：患者标识交叉引用管理器”模式的使用约束

要使用“Healthcare：患者标识交叉引用管理器”模式，必须满足以下约束：

  • 必须将受支持版本的 IBM® InfoSphere® Master Data Management (MDM) 用作您的 EMPI。有关受支持版本的更多信息，请在 IBM Integration Bus Healthcare Pack 文档中，转至“IBM Integration Bus Healthcare Pack 的平台支持”并单击“IBM Integration Bus 系统需求”。
  • 在 IBM InfoSphere MDM 中，您的“主数据管理项目”必须使用“患者”模板。
  • 您必须对要连接到的每个“患者标识源”参与者都使用唯一模式实例。
