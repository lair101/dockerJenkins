「患者 ID 相互参照マネージャー」パターンを適用する前に完了するタスク

パターンを使用してパターン・インスタンスを生成する前に、必要に応じて、以下の 1 つ以上のタスクを完了します。

  • 「患者 ID 相互参照マネージャー」パターンがビジネス要件に適合していることを確認します。
  • IBM® InfoSphere® Master Data Management (MDM) を EMPI としてインストールして構成します。以下の項目を構成しておきます。
    • マスター・データ管理プロジェクトでは患者テンプレートを使用する必要があります。
    • EMPI の患者 ID ドメインがセットアップされていなければならず、対応する srcCode を定義する必要があります。
  • EMPI によって使用される Web サービスの URL、および EMPI への接続に必要なユーザー名とパスワードを指定します。
