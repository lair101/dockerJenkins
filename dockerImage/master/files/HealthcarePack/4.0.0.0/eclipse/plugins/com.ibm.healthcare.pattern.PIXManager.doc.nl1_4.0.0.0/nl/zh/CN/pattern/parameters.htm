“患者标识交叉引用管理器”模式的参数

连接详细信息
模式参数 缺省值 描述
连接详细信息 localhost:1111

该参数定义“患者标识交叉引用管理器”的 TCP/IP 连接详细信息。请配置在 PIX 概要文件中充当“患者标识源”的临床应用程序以连接到该地址。该参数的格式为 hostname:port。
MDM 患者中心 (EMPI)
模式参数 缺省值 描述
用户名

该参数指定连接到 EMPI 所需的用户名。
密码

该参数指定连接到 EMPI 所需的密码。
Web 服务 URL http://localhost:9080/ibminitiatews/services/IdentityHub

该参数指定连接到 EMPI 标识中心所需的 Web 服务 URL，例如：http://localhost:9080/ibminitiatews/services/IdentityHub。
PIX 管理器
模式参数 缺省值 描述
应用程序

该参数定义在从“患者标识交叉引用管理器”所发送应答的 MSH 分段中输入的“发送应用程序”值。
设施

该参数定义在从“患者标识交叉引用管理器”所发送应答的 MSH 分段中输入的“发送工具”值。
名称空间标识

该参数定义“患者标识交叉引用管理器”的分配中心（域）的名称空间标识。该参数等同于 EMPI 中的 srcCode。

在入局消息中，如果 PID.3.4.1 字段的值与该参数的值不匹配，那么该消息将被拒绝。在入局消息中，如果 PID.3.4.1 为空，但 PID.3.4.2 和 PID.3.4.3 与其对应的模式参数（“通用标识”和“通用标识类型”）匹配，那么将用该参数的值来填充 PID.3.4.1。
通用标识

该参数定义“患者标识交叉引用管理器”的分配中心（域）的通用标识。

在入局消息中，如果 PID.3.4.2 字段的值与该参数的值不匹配，那么该消息将被拒绝。
通用标识类型 ISO

该参数定义“患者标识交叉引用管理器”的分配中心（域）的通用标识类型。

在入局消息中，如果 PID.3.4.3 字段的值与该参数的值不匹配，那么该消息将被拒绝。
患者身份源
模式参数 缺省值 描述
应用程序

该参数定义入局消息的 MSH.3 字段的期望“发送应用程序”值。

如果消息中的值与为该参数定义的值不匹配，那么入局消息将被拒绝。
设施

该参数定义入局消息的 MSH.4 字段的期望“发送工具”值。

如果消息中的值与为该参数定义的值不匹配，那么入局消息将被拒绝。

所有模式参数均随其缺省值一起列出。 未指示缺省值时，表明缺省值是空字符串。
