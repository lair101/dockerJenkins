Parámetros de patrón para el patrón Patient Identifier Cross-reference Manager

Detalles de conexión
Parámetro de patrón Valor predeterminado Descripción
Información detallada de conexión localhost:1111

Este parámetro define los detalles de la conexión TCP/IP del Patient Identifier Cross-reference Manager. Configure su aplicación clínica que actúa como Origen de identidad de pacientes en el perfil PIX para que se conecte a esta dirección. El parámetro tiene el formato nombre_host:puerto
MDM Patient Hub (EMPI)
Parámetro de patrón Valor predeterminado Descripción
Nombre de usuario

Este parámetro especifica el nombre de usuario que se necesita para conectarse al EMPI.
Contraseña

Este parámetro especifica la contraseña que se necesita para conectarse al EMPI.
URL de servicio web http://localhost:9080/ibminitiatews/services/IdentityHub

Este parámetro especifica el URL de servicio web que se necesita para conectar con el concentrador de identidades EMPI, por ejemplo: http://localhost:9080/ibminitiatews/services/IdentityHub
Gestor de PIX
Parámetro de patrón Valor predeterminado Descripción
Aplicación

Este parámetro define el valor de aplicación emisora que se especifica en el segmento MSH de los acuses de recibo que se envían desde el Patient Identifier Cross-reference Manager.
Recurso

Este parámetro define el valor de recurso emisor que se especifica en el segmento MSH de los acuses de recibo que se envían desde el Patient Identifier Cross-reference Manager.
ID de espacio de nombres

Este parámetro define el ID de espacio de nombres de la autoridad de asignación (dominio) del Patient Identifier Cross-reference Manager. Este parámetro es el equivalente de srcCode en el EMPI.

En un mensaje entrante, si el valor del campo PID.3.4.1 no coincide con el valor de este parámetro, se rechaza el mensaje. En un mensaje entrante, si PID.3.4.1 está vacío pero PID.3.4.2 y PID.3.4.3 coinciden con sus respectivos parámetros de patrón (ID Universal y Tipo de ID Universal), PID.3.4.1 se cumplimenta con el valor de este parámetro.

ID universal

Este parámetro define el ID universal de la autoridad de asignación (dominio) del Patient Identifier Cross-reference Manager.

En un mensaje entrante, si el valor del campo PID.3.4.2 no coincide con el valor de este parámetro, se rechaza el mensaje.
Tipo de ID universal ISO

Este parámetro define el tipo de ID universal de la autoridad de asignación (dominio) del Patient Identifier Cross-reference Manager.

En un mensaje entrante, si el valor del campo PID.3.4.3 no coincide con el valor de este parámetro, se rechaza el mensaje.
Origen de identidad de pacientes
Parámetro de patrón Valor predeterminado Descripción
Aplicación

Este parámetro define el valor esperado para Aplicación emisora del campo MSH.3 del mensaje entrante.

Si el valor que hay en el mensaje no coincide con el valor definido para este parámetro, se rechaza el mensaje entrante.
Recurso

Este parámetro define el valor esperado para Recurso emisor del campo MSH.4 del mensaje entrante.

Si el valor que hay en el mensaje no coincide con el valor definido para este parámetro, se rechaza el mensaje entrante.

Todos los parámetros de patrón aparecen en la lista con sus valores predeterminados. Cuando no se indica un valor predeterminado, éste es una serie vacía.
