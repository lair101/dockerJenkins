Bedingungen für die Verwendung des Musters Healthcare: Patient Identifier Cross-reference Manager

Damit Sie das Muster Healthcare: Patient Identifier Cross-reference Manager verwenden können, müssen Sie die folgenden Bedingungen erfüllen:

  • Sie müssen eine unterstützte Version von IBM® InfoSphere® Master Data Management (MDM) als Ihren EMPI verwenden. Weitere Informationen zu unterstützten Versionen finden Sie in der Dokumentation zu IBM Integration Bus Healthcare Pack im Abschnitt "Plattformunterstützung für IBM Integration Bus Healthcare Pack" unter "Systemvoraussetzungen für IBM Integration Bus".
  • In IBM InfoSphere MDM muss Ihr Master Data Management Project die Vorlage Patient verwenden.
  • Sie müssen für jeden Patient Identity Source-Akteur, zu dem Sie eine Verbindung herstellen möchten, eine eindeutige Musterinstanz verwenden.
