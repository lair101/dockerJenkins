Tasks, die vor der Anwendung des Musters Patient Identifier Cross-reference Manager auszuführen sind

Bevor Sie das Muster verwenden und eine Musterinstanz generieren, müssen Sie unter Umständen eine oder auch mehrere der folgenden Tasks ausführen:

  • Vergewissern Sie sich, dass das Muster Patient Identifier Cross-reference Manager Ihren Geschäftsanforderungen entspricht.
  • Installieren und konfigurieren Sie IBM® InfoSphere® Master Data Management (MDM) als Ihren EMPI. Vergewissern Sie sich, dass die folgenden Elemente konfiguriert sind:
    • Ihr Master Data Management Project muss die Vorlage Patient verwenden.
    • Die Domäne für Patientenidentifikation für Ihren EMPI muss konfiguriert werden und Sie müssen einen entsprechenden srcCode definieren.
  • Geben Sie die Web-Service-URL an, die von Ihrem EMPI verwendet wird, sowie die Kombination aus Benutzername und Kennwort, die für die Verbindung zum EMPI erforderlich ist.
