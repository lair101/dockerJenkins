Parameter für das Muster Patient Identifier Cross-reference Manager

Verbindungsdetails
Musterparameter Standardeinstellung Beschreibung
Verbindungsdetails localhost:1111

Dieser Parameter definiert die Details der TCP/IP-Verbindung des Patient Identifier Cross-reference Manager. Konfigurieren Sie Ihre klinische Anwendung, die als Patient Identity Source im PIX-Profil agiert, dahingehend, dass sie eine Verbindung zu dieser Adresse herstellt. Dieser Parameter hat das Format Hostname:Port.

MDM Patient Hub (EMPI)
Musterparameter Standardeinstellung Beschreibung
Benutzername

Dieser Parameter gibt den Benutzernamen an, der für die Verbindung zu dem EMPI erforderlich ist.
Kennwort

Dieser Parameter gibt das Kennwort an, das für die Verbindung zu dem EMPI erforderlich ist.
Web-Service-URL http://localhost:9080/ibminitiatews/services/IdentityHub

Dieser Parameter gibt die Web-Service-URL an, die für die Verbindung zum EMPI-Identitätshub erforderlich ist, z. B.: http://localhost:9080/ibminitiatews/services/IdentityHub.
PIX Manager
Musterparameter Standardeinstellung Beschreibung
Anwendung

Dieser Parameter definiert den Wert für die sendende Anwendung, der im MSH-Segment der vom Patient Identifier Cross-reference Manager gesendeten Bestätigungen (ACK) eingegeben wird.
Einrichtung

Dieser Parameter definiert den Wert für die sendende Einrichtung, der im MSH-Segment der vom Patient Identifier Cross-reference Manager gesendeten Bestätigungen eingegeben wird.
Namespace-ID

Dieser Parameter definiert die Namespace-ID der zuweisenden Autorität (Domäne) des Patient Identifier Cross-reference Manager. Dieser Parameter entspricht funktional dem Parameter srcCode im EMPI.

Wenn der Wert des Felds "PID.3.4.1" in einer eingehenden Nachricht nicht mit dem Wert dieses Parameters übereinstimmt, wird die Nachricht zurückgewiesen. Wenn das Feld "PID.3.4.1" leer ist, die Werte der Felder "PID.3.4.2" und "PID.3.4.3" jedoch ihren entsprechenden Musterparametern ("Universal-ID" und "Universal ID Type) entsprechen, wird im Feld "PID.3.4.1" der Wert dieses Parameters eingetragen.

Universal-ID

Dieser Parameter definiert die Universal-ID der zuweisenden Autorität (Domäne) des Patient Identifier Cross-reference Manager.

Wenn der Wert des Felds "PID.3.4.2" in einer eingehenden Nachricht nicht mit dem Wert dieses Parameters übereinstimmt, wird die Nachricht zurückgewiesen.

Universal-ID-Typ ISO

Dieser Parameter definiert den Universal-ID-Typ der zuweisenden Autorität (Domäne) des Patient Identifier Cross-reference Manager.

Wenn der Wert des Felds "PID.3.4.3" in einer eingehenden Nachricht nicht mit dem Wert dieses Parameters übereinstimmt, wird dieNachricht zurückgewiesen.

Patient Identity Source
Musterparameter Standardeinstellung Beschreibung
Anwendung

Dieser Parameter definiert den erwarteten Wert für die sendende Anwendung des Felds "MSH.3" der eingehenden Nachricht.

Wenn der Wert in der Nachricht nicht mit dem für diesen Parameter definierten Wert übereinstimmt, wird die eingehende Nachricht zurückgewiesen.

Einrichtung

Dieser Parameter definiert den erwarteten Wert für die sendende Einrichtung des Felds "MSH.4" der eingehenden Nachricht.

Wenn der Wert in der Nachricht nicht mit dem für diesen Parameter definierten Wert übereinstimmt, wird die eingehende Nachricht zurückgewiesen.

Für alle Musterparameter sind die Standardwerte aufgeführt. Ist kein Standardwert angegeben, ist der Standardwert eine leere Zeichenfolge.
