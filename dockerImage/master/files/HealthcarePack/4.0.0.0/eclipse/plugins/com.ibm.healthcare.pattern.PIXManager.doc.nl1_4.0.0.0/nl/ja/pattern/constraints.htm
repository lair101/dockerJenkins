「Healthcare: Patient Identifier Cross-reference Manager」パターンの使用に関する制約

「Healthcare: Patient Identifier Cross-reference Manager」パターンを使用するには、以下の制約に対処する必要があります。

  • サポートされるバージョンの IBM® InfoSphere® Master Data Management (MDM) を EMPI として使用する必要があります。 サポートされるバージョンについて詳しくは、IBM Integration Bus Healthcare Pack 文書内の「Platform support for IBM Integration Bus Healthcare Pack」にアクセスして、「IBM Integration Bus system requirements」をクリックしてください。
  • IBM InfoSphere MDM では、マスター・データ管理プロジェクトで患者テンプレートを使用する必要があります。
  • 接続先の患者 ID ソース・アクターごとに、固有のパターン・インスタンスを使用する必要があります。
