Restricciones de uso del patrón Servicio sanitario: Patient Identifier Cross-reference Manager

Para utilizar el patrón Servicio sanitario: Patient Identifier Cross-reference Manager, debe cumplir las siguientes restricciones:

  • Debe utilizar una versión soportada de IBM® InfoSphere® Master Data Management (MDM) como su EMPI. Para obtener más información sobre las versiones soportadas, vaya a "Soporte de plataforma para IBM Integration Bus Healthcare Pack" en la documentación de IBM Integration Bus Healthcare Pack y pulse en "Requisitos de sistema de IBM Integration Bus".
  • En IBM InfoSphere MDM, su Proyecto de gestión de datos maestros debe utilizar la plantilla Paciente.
  • Debe utilizar una instancia de patrón exclusiva para cada actor de origen de identidad pacientes al que desea conectarse.
