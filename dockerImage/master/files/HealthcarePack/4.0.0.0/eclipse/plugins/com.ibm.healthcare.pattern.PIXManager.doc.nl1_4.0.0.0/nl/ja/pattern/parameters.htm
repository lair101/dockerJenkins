「患者 ID 相互参照マネージャー」パターンのパラメーター

接続の詳細
パターン・パラメーター デフォルト 説明
接続の詳細 localhost:1111

このパラメーターは、患者 ID 相互参照マネージャーの TCP/IP 接続の詳細を定義します。PIX プロファイル内の患者 ID ソースとなる臨床アプリケーションを、このアドレスに接続するように構成します。このパラメーターは hostname:port 形式です。
MDM 患者ハブ (EMPI)
パターン・パラメーター デフォルト 説明
ユーザー名

このパラメーターは、EMPI に接続するために必要なユーザー名を指定します。
パスワード

このパラメーターは、EMPI に接続するために必要なパスワードを指定します。
Web サービス URL http://localhost:9080/ibminitiatews/services/IdentityHub

このパラメーターは、EMPI ID ハブに接続するために必要な Web サービス URL を指定します (例: http://localhost:9080/ibminitiatews/services/IdentityHub)。
PIX マネージャー
パターン・パラメーター デフォルト 説明
アプリケーション

このパラメーターは、患者 ID 相互参照マネージャーから送信される確認応答の MSH セグメントに入れられる「送信側アプリケーション」値を定義します。
ファシリティー

このパラメーターは、患者 ID 相互参照マネージャーから送信される確認応答の MSH セグメントに入れられる「送信側ファシリティー」値を定義します。
ネーム・スペース ID

このパラメーターは、患者 ID 相互参照マネージャーの割り当て権限 (ドメイン) のネーム・スペース ID を定義します。このパラメーターは、EMPI での srcCode に相当します。

着信メッセージでは、PID.3.4.1 フィールドの値がこのパラメーターの値と一致しなければ、メッセージは拒否されます。 着信メッセージでは、PID.3.4.1 が空でも PID.3.4.2 と PID.3.4.3 がそれぞれの対応するパターン・パラメーター (ユニバーサル ID とユニバーサル ID タイプ) と一致すれば、このパラメーターの値が PID.3.4.1 に取り込まれます。

ユニバーサル ID

このパラメーターは、患者 ID 相互参照マネージャーの割り当て権限 (ドメイン) のユニバーサル ID を定義します。

着信メッセージでは、PID.3.4.2 フィールドの値がこのパラメーターの値と一致しなければ、メッセージは拒否されます。

ユニバーサル ID タイプ ISO

このパラメーターは、患者 ID 相互参照マネージャーの割り当て権限 (ドメイン) のユニバーサル ID タイプを定義します。

着信メッセージでは、PID.3.4.3 フィールドの値がこのパラメーターの値と一致しなければ、メッセージは拒否されます。

患者 ID ソース
パターン・パラメーター デフォルト 説明
アプリケーション

このパラメーターは、着信メッセージの MSH.3 フィールドの「送信側アプリケーション」予期値を定義します。

メッセージ内の値がこのパラメーターに定義した値と一致しなければ、着信メッセージは拒否されます。
ファシリティー

このパラメーターは、着信メッセージの MSH.4 フィールドの「送信側ファシリティー」予期値を定義します。

メッセージ内の値がこのパラメーターに定義した値と一致しなければ、着信メッセージは拒否されます。

すべてのパターン・パラメーターとそのデフォルト値をリストします。 デフォルトが示されていない場合、デフォルト値は空ストリングです。
