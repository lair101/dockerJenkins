Parameters for the Healthcare: Home Health pattern

SOAP_Input
Pattern parameter Default Description
Path suffix for URL /DeviceObservationConsumer

This property identifies the location from where web service requests are retrieved. Do not use the full URL. If the URL that you want is http://hostname[:port]/[path], specify either /path or /path fragment/* where * is a wildcard that you can use to mean match any.

Security
Pattern parameter Default Description
Include SAML 2.0 token pass-through Selected

Checking this box will include a PEP node to enable external authorization and authentication of a SAML 2.0 token

All the pattern parameters are listed with their default values. Where no default is indicated, the default value is an empty string.
