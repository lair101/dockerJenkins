Datenbank der Patienten-IDs bereinigen

Sobald Sie das MedicalMedicalDeviceInput-Knotenbeispiel abgeschlossen haben, können Sie die Datenbank, deren zugehörige ODBC-Definition sowie die Werte für die Benutzer-ID und das Kennwort, die dem Datenquellennamen zugeordnet sind, entfernen.

Führen Sie zur Bereinigung der Datenbankumgebung die folgenden Tasks aus:

  1. Entfernen Sie die Datenbank und die ODBC-Definition.

    Befolgen Sie die für Ihren Datenbankmanager relevanten Anweisungen:

    • DB2 bereinigen
    • Oracle bereinigen
  2. Entfernen Sie die Werte für die Benutzer-ID und das Kennwort, die dem Datenquellennamen PATIENTSDSN zugeordnet sind.

    Öffnen Sie ein Befehlskonsolenfenster, in dem Sie IBM Integration Bus-Befehle ausführen können. Geben Sie zum Entfernen der Benutzer-ID und des Kennworts den folgenden Befehl ein: 

    mqsisetdbparms Integrationsknoten -n PATIENTSDSN -d
    3. Dabei ist Integrationsknoten der Name Ihres Integrationsknotens.
