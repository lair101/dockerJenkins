「Healthcare: Medical devices to EMR」パターンのリソース

注: このページ内のリンクは、IBM Integration Toolkit のパターン・エクスプローラーからパターンにアクセスするときのみ使用できます。

「Healthcare: Medical devices to EMR」パターンでは、HL7 メッセージを解析、検証、および変換するためのメッセージ・セットを使用します。

以下のリンクをクリックすることにより、HL7 メッセージ・セットおよびテスト・リソースをインポートできます。

  • HL7 リソースをワークスペースにインポートします

サンプル・リソースを使用し終わったら、それらをワークスペースから削除します。

「Healthcare: Medical devices to EMR」パターンによって作成される Web サービスでは、SOAP ノードを構成するためのメッセージ・セットを使用します。 このパターンでは、MedicalDeviceInput ノードからの測定をモデル化するためのメッセージ・セットも提供されます。

以下のリンクをクリックすることにより、これらのメッセージ・セットをインポートできます。

  • メッセージ・セットをワークスペースにインポートします

サンプル・リソースを使用し終わったら、それらをワークスペースから削除します。

以下のリンクをクリックすることにより、GenericHL7Input ノードおよび GenericHL7Output ノードのソース・コードをインポートできます。

  • ソース・コードをワークスペースにインポートします

ソース・コード・リソースの使用を終えたら、それらをワークスペースから削除します。
