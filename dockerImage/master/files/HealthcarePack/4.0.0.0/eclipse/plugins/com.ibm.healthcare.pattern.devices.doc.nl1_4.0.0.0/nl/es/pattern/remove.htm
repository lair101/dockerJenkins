Limpieza de la base de datos de identificadores de pacientes

Cuando haya acabado con el ejemplo de nodo MedicalMedicalDeviceInput, puede eliminar la base de datos, su definición de ODBC y los valores de identificador de usuario y contraseña asociados al nombre del origen de datos.

Lleve a cabo las tareas siguientes para limpiar el entorno de base de datos:

  1. Elimine la base de datos y la definición ODBC.

    Siga las instrucciones relevantes para su gestor de base de datos:

    • Limpieza de DB2
    • Limpieza de Oracle
  2. Elimine el identificador de usuario y la contraseña asociados al nombre del origen de datos PATIENTSDSN .

    Abra una ventana de consola de mandatos donde pueda ejecutar mandatos de IBM Integration Bus. Especifique el mandato siguiente para eliminar el identificador de usuario y la contraseña: 

    mqsisetdbparms nodo_integración -n
PATIENTSDSN -d
    3. donde nodo_integración es el nombre del nodo de integración.
