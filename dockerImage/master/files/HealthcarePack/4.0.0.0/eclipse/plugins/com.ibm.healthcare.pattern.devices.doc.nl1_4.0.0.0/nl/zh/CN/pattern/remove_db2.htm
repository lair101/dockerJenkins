清理 DB2

下列指示信息假定您使用的是 DB2 通用数据库，并指导您完成以下任务：

  • 除去此模式所使用的名为 PATSDB 的数据库
  • 除去此数据库的 ODBC 定义

要除去此数据库和 ODBC 定义，请完成下列步骤：

  1. 在 DB2 命令窗口中，输入以下命令以确保 DB2 已启动：
    db2start
  2. 在 DB2 命令窗口中，输入以下命令：
    db2 DROP DATABASE PATSDB
  3. 要除去 PATSDB 数据库的 ODBC 定义（其数据源名称为 PATIENTSDSN），请撤销 ODBC 设置步骤；请参阅 IBM Integration Bus 文档中的“启用与数据库的 ODBC 连接”。
