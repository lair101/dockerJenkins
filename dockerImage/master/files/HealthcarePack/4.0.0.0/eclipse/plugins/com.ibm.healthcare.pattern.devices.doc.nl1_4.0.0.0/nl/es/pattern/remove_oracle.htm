Limpieza de Oracle

Estas instrucciones dan por supuesto que utiliza DB2 Universal Database y le guían por las tareas siguientes:

  • Elimine la base de datos denominada PATSDB que utiliza el patrón
  • Elimine la definición ODBC de la base de datos

Para eliminar la base de datos y la definición ODBC:

  1. En una ventana de mandatos, entre el siguiente mandato: 
    sqlplus <uid>/<password>
DROP DATABASE PATSDB;
  2. Para eliminar la definición ODBC de la base de datos PATSDB, (cuyo nombre de origen de datos es PATIENTSDSN), deshaga el paso de configuración se ODBC; consulte Habilitación de las conexiones ODBC con las bases de datos en la documentación de IBM Integration Bus.
