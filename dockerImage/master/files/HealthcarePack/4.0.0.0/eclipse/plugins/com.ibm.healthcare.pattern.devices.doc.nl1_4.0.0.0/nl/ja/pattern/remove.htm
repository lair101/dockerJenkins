患者 ID データベースのクリーンアップ

MedicalDeviceInput ノード・サンプルの使用を終えたら、データベースと、その ODBC 定義と、データ・ソース名に関連付けられたユーザー ID およびパスワードの値とを削除することができます。

データベース環境をクリーンアップするには、以下のタスクを実行します。

  1. データベースおよび ODBC 定義を削除します。

    ご使用のデータベース・マネージャーに該当する手順に従ってください。

    • DB2 のクリーンアップ
    • Oracle のクリーンアップ
  2. データ・ソース名 PATIENTSDSN に関連付けられているユーザー ID とパスワードを削除します。

    IBM Integration Bus コマンドを実行できる、コマンド・コンソール・ウィンドウを開きます。 以下のコマンドを入力して、ユーザー ID およびパスワードを削除します。

    mqsisetdbparms integration_node -n PATIENTSDSN -d
    3. integration_node は統合ノードの名前です。
