Ressourcen für das Muster 'Healthcare: Medizinische Geräte an EKA'

Hinweis: Die Links auf dieser Seite können nur verwendet werden, wenn Sie über den Musterexplorer im IBM Integration Toolkit auf das Muster zugreifen.

Das Muster Healthcare: Medizinische Geräte an EKA analysiert, überprüft und transformiert HL7-Nachrichten mithilfe einer Nachrichtengruppe.

Sie können die HL7-Nachrichtengruppe und Testressourcen importieren, indem Sie auf folgenden Link klicken:

  • HL7-Ressourcen in Ihren Arbeitsbereich importieren

Wenn Sie mit der Verarbeitung der Beispielressourcen fertig sind, löschen Sie diese aus Ihrem Arbeitsbereich.

Der vom Muster Healthcare: Medizinische Geräte an EKA erstellte Web-Service verwendet für die Konfiguration der SOAP-Knoten eine Nachrichtengruppe. Das Muster bietet außerdem eine Nachrichtengruppe für die Modellierung der Messungen vom MedicalDeviceInput-Knoten.

Sie können diese Nachrichtengruppen importieren, indem Sie auf folgenden Link klicken:

  • Nachrichtengruppen in Ihren Arbeitsbereich importieren

Wenn Sie mit der Verarbeitung der Beispielressourcen fertig sind, löschen Sie diese aus Ihrem Arbeitsbereich.

Sie können den Quellcode für die GenericHL7Input- und GenericHL7Output-Knoten importieren, indem Sie auf folgenden Link klicken:

  • Quellcode in Ihren Arbeitsbereich importieren

Wenn Sie mit der Verarbeitung der Quellcoderessourcen fertig sind, löschen Sie diese aus Ihrem Arbeitsbereich.
