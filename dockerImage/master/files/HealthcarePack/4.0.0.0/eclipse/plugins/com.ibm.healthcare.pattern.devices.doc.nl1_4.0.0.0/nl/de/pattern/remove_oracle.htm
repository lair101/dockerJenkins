Oracle bereinigen

Bei diesen Anweisungen wird vorausgesetzt, dass Sie DB2 Universal Database verwenden. Sie werden durch die folgenden Tasks geführt:

  • Entfernen der vom Muster verwendeten Datenbank PATSDB
  • Entfernen der ODBC-Definition für die Datenbank

So entfernen Sie die Datenbank und die ODBC-Definition:

  1. Geben Sie in einem Befehlsfenster folgenden Befehl ein: 
    sqlplus <Benutzer-ID>/<Kennwort>
DROP DATABASE PATSDB;
  2. Um die ODBC-Definition für die PATSDB-Datenbank (mit dem Datenquellennamen PATIENTSDSN) zu entfernen, müssen Sie die ODBC-Konfiguration rückgäng machen; Informationen hierzu finden Sie im Abschnitt 'ODBC-Verbindungen zu den Datenbanken herstellen' in der Dokumentation zu IBM Integration Bus.
