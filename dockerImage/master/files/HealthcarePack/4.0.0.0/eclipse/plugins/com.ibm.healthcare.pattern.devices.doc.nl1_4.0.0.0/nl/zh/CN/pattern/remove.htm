清理患者标识数据库

完成使用“MedicalMedicalDeviceInput 节点”样本后，您可以除去数据库、其 ODBC 定义以及与数据源名称相关联的用户标识和密码值。

要清理数据库环境，请完成以下任务：

  1. 除去数据库和 ODBC 定义。

    按照与数据库管理器相关的指示信息执行操作：

    • 清理 DB2
    • 清理 Oracle
  2. 除去与数据源名称 PATIENTSDSN 相关联的用户标识和密码。

    打开命令控制台窗口，您可以从中运行 IBM Integration Bus 命令。输入以下命令以除去用户标识和密码：

    mqsisetdbparms integration_node -n PATIENTSDSN -d
    3. 其中 integration_node 是集成节点的名称。
返回到“Healthcare：医疗设备到 EMR”模式规范
