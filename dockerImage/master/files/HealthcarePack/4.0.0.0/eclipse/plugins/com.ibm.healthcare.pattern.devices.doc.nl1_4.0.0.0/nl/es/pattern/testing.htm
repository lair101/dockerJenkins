Prueba del patrón Servicio sanitario: Dispositivos médicos a EMR

Utilice la información siguiente para probar el patrón Servicio sanitario: Dispositivos médicos a EMR.

Configuración de la instancia de patrón básica

Los flujos de mensajes principales de la instancia de patrón son:

  • MedicalDevices
  • TransformAndRoute
  • Sender
  • PatientIdentifiers

Debe asegurarse de que tiene todas las colas configuradas para la instancia de patrón, consulte la sección "Colas" en Gestión de una instancia de patrón Servicio sanitario: Dispositivos médicos a EMR.

Flujo de información a través de la instancia de patrón

Cuando se despliega una instancia del patrón Servicio sanitario: Dispositivos médicos a EMR, la información fluye en la secuencia siguiente.

  1. El nodo MedicalDeviceInput localiza su servicio configurable para determinar a los dispositivos médicos a los que debe conectarse y las mediciones que debe recopilar.
  2. El nodo MedicalDeviceInput lee una medición de un dispositivo médico y propaga la información en forma de mensaje en el flujo de mensajes MedicalDevices.
  3. El flujo de mensajes MedicalDevices añade un tiempo de caducidad y envía el mensaje a la cola RXF del flujo de mensajes TransformAndRoute.
  4. El flujo de mensajes TransformAndRoute convierte el mensaje de dispositivo en un resultado de observación HL7. El mensaje se publica en formato canónico HL7 para que otras aplicaciones pueden consumirlo. El resultado de observación de asocia al identificador actual de paciente para ese dispositivo y después se graba en la cola DEST.
  5. El flujo de mensajes Sender envía el mensaje HL7 al EMR, ce responde con un ACK.
  6. El flujo Sender procesa el mensaje ACK.

Todos los flujos tienen también un subflujo de manejo de excepciones que maneja los problemas que capte el flujo.

La aplicación de prueba de HL7 se puede utilizar durante el desarrollo para simular que se recibe el EMR. Inicie la aplicación de prueba e inicie un escucha MLLP TCP/IP. La aplicación de prueba recibirá y mostrará los resultados de observación que envíe la instancia de patrón.

Para iniciar la aplicación de prueba de HL7, realice los pasos siguientes:

  1. Pulse Archivo > Nuevo > Otro.
  2. En la carpeta Healthcare, seleccione Aplicación de prueba de HL7 y pulse Siguiente.
  3. Especifique un contenedor y nombre de archivo para los valores de la aplicación de prueba y pulse Finalizar. La aplicación de prueba de HL7 se inicia.
