Oracle のクリーンアップ

ここで示す手順では、DB2 Universal Database を使用することを前提に、以下のタスクについて説明します。

  • パターンで使用されている PATSDB というデータベースを削除します。
  • データベースの ODBC 定義を削除します。

データベースおよび ODBC 定義を削除するには、以下のとおりにします。

  1. コマンド・ウィンドウで、以下のコマンドを入力します。 
    sqlplus <uid>/<password>
DROP DATABASE PATSDB;
  2. PATSDB データベース (データ・ソース名は PATIENTSDSN) の ODBC 定義を削除するには、ODBC セットアップ・ステップでの設定を元に戻します。 IBM Integration Bus の資料で『データベースへの ODBC 接続の使用可能化』を参照してください。
