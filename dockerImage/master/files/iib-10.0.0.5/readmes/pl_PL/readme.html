Instalowanie pakietów języków narodowych dla pakietu IBM Integration Toolkit

Objaw:
W wersji 10 instalacja produktu IBM Integration Bus nie obejmuje pakietów językowych dla pakietu IBM Integration Toolkit. Domyślnie interfejs pakietu IBM Integration Toolkit jest wyświetlany w języku angielskim.

Rozwiązanie:
Należy pobrać składnik Pakiet językowy pakietu IBM Integration Toolkit.

Składnik Pakiet językowy pakietu IBM Integration Toolkit zawiera pakiety językowe dla następujących języków:

  • Niemiecki, hiszpański, francuski, włoski, japoński, koreański, portugalski (Brazylia), chiński uproszczony, chiński tradycyjny (Pakiet językowy 1)
  • Polski, rosyjski (Pakiet językowy 2)
  • Turecki (Pakiet językowy 2a)

Aby zainstalować składnik Pakiet językowy pakietu IBM Integration Toolkit, wykonaj następujące czynności:

  1. W przeglądarce WWW przejdź do następującego adresu URL: ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. Pobierz odpowiedni składnik Pakiet językowy.
    Uwaga: Należy wybrać wersję składnika Pakiet językowy pakietu IBM Integration Toolkit, która jest zgodna z wersją produktu IBM Integration Bus.
  3. Uruchom pakiet IBM Integration Toolkit.
  4. W menu kliknij opcję Help > Install New Software (Pomoc > Zainstaluj nowe oprogramowanie). Zostanie otwarte okno Available Software (Dostępne oprogramowanie).
  5. Kliknij opcję Add (Dodaj), a następnie wpisz nazwę repozytorium. Na przykład: Pakiety językowe.
  6. Kliknij opcję Archive (Archiwum), przejdź do miejsca, do którego został pobrany składnik Pakiet językowy pakietu IBM Integration Toolkit i kliknij przycisk OK. Zostanie wyświetlony składnik Pakiet językowy pakietu IBM Integration Toolkit.
  7. Rozwiń składnik Pakiet językowy pakietu IBM Integration Toolkit, aby wyświetlić poszczególne pakiety językowe, zaznacz pola wyboru obok języków, które mają zostać zainstalowane, a następnie kliknij przycisk Next (Dalej).
  8. Ponownie kliknij przycisk Next (Dalej), kliknij, aby zaakceptować umowę licencyjną, a następnie kliknij przycisk Finish (Zakończ). Wybrane pakiety językowe zostaną zainstalowane, a użytkownik zostanie poproszony o zrestartowanie pakietu IBM Integration Toolkit.
  9. Kliknij przycisk Yes (Tak), aby zrestartować pakiet IBM Integration Toolkit. Jeśli w systemie operacyjnym są używane ustawienia narodowe zgodne z jednym z języków obsługiwanych przez zainstalowane pakiety językowe, interfejs użytkownika pakietu IBM Integration Toolkit zostanie wyświetlony w tym języku.
