Installazione dei pacchetti linguistici per IBM Integration Toolkit

Sintomo:
nella versione 10, l'installazione di IBM Integration Bus non include i pacchetti linguistici per IBM Integration Toolkit. Per impostazione predefinita, l'interfaccia di IBM Integration Toolkit viene visualizzata in inglese.

Soluzione:
Scaricare la funzione IBM Integration Toolkit Language Pack.

La funzione IBM Integration Toolkit Language Pack contiene i pacchetti linguistici per le seguenti lingue:

  • Tedesco, spagnolo, francese, italiano, giapponese, coreano, portoghese brasiliano, cinese semplificato, cinese tradizionale (Language Pack 1)
  • Polacco, russo (Language Pack 2)
  • Turco (Language Pack 2a)

Per installare una funzione IBM Integration Toolkit Language Pack, completare la seguente procedura:

  1. In un browser web, passare al seguente URL: ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. Scaricare la funzione Language Pack pertinente.
    Nota: accertarsi di selezionare la versione di IBM Integration Toolkit Language Pack corrispondente alla propria versione di IBM Integration Bus.
  3. Avviare IBM Integration Toolkit.
  4. Dal menu, fare clic su Guida > Installa nuovo software. Viene visualizzata la finestra Software disponibile.
  5. Fare clic su Aggiungi e immettere un nome per il repository. Ad esempio: Language Pack.
  6. Fare clic su Archivio, passare alla posizione in cui è stata scaricata la funzione IBM Integration Toolkit Language Pack e fare clic su OK. Viene visualizzata la funzione IBM Integration Toolkit Language Pack.
  7. Espandere la funzione IBM Integration Toolkit Language Pack per visualizzare i singoli pacchetti linguistici, selezionare le caselle di spunta relative ai pacchetti linguistici da installare e fare clic su Avanti.
  8. Fare di nuovo clic su Avanti, accettare l'accordo di licenza e fare clic su Fine. I pacchetti linguistici selezionati vengono installati e verrà richiesto di riavviare IBM Integration Toolkit.
  9. Fare clic su Sì per riavviare IBM Integration Toolkit. Se la locale del sistema operativo è configurata per una delle lingue supportate dai pacchetti linguistici installati, l'interfaccia utente di IBM Integration Toolkit viene visualizzata in tale lingua.
