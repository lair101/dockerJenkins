Landessprachenpakete für IBM Integration Toolkit installieren

Symptom:
In Version 10 umfasst die Installation von IBM Integration Bus die Sprachenpakete für IBM Integration Toolkit nicht. Standardmäßig wird die IBM Integration Toolkit-Schnittstelle in Englisch angezeigt.

Lösung:
Laden Sie das Feature für die IBM Integration Toolkit-Sprachenpakete herunter.

Das Feature für die IBM Integration Toolkit-Sprachenpakete enthält Sprachenpakete für die folgenden Sprachen:

  • Deutsch, Spanisch, Französisch, Italienisch, Japanisch, Koreanisch, Portugiesisch (Brasilien), vereinfachtes Chinesisch, traditionelles Chinesisch (Sprachenpaket 1)
  • Polnisch, Russisch (Sprachenpaket 2)
  • Türkisch (Sprachenpaket 2a)

Führen Sie zum Installieren eines IBM Integration Toolkit-Sprachenpaketfeatures die folgenden Schritte aus:

  1. Navigieren Sie in einem Web-Browser zur folgenden URL: ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. Laden Sie das entsprechende Integration Toolkit-Sprachenpaketfeature herunter.
    Anmerkung: Stellen Sie sicher, dass Sie die Version des IBM Integration Toolkit-Sprachenpaketfeatures auswählen, die Ihrer Version von IBM Integration Bus entspricht.
  3. Starten Sie IBM Integration Toolkit.
  4. Klicken Sie im Menü auf Hilfe > Neue Software installieren. Das Fenster "Verfügbare Software" wird geöffnet.
  5. Klicken Sie auf Hinzufügen und geben Sie einen Namen für das Repository ein, z. B. Sprachenpakete.
  6. Klicken Sie auf Archivieren, navigieren Sie zu der Position, an die Sie das IBM Integration Toolkit-Sprachenpaketfeature heruntergeladen haben, und klicken Sie dann auf OK. Das IBM Integration Toolkit-Sprachenpaketfeature wird angezeigt.
  7. Erweitern Sie das IBM Integration Toolkit-Sprachenpaketfeature, um die einzelnen Sprachenpakete anzuzeigen, wählen Sie die Kontrollkästchen für die Sprachenpakete aus, die Sie installieren möchten, und klicken Sie dann auf Weiter.
  8. Klicken Sie erneut auf Weiter, klicken Sie auf das Kontrollkästchen zum Akzeptieren der Lizenzvereinbarung und klicken Sie dann auf Fertigstellen. Die ausgewählten Sprachenpakete werden installiert, und Sie werden aufgefordert, IBM Integration Toolkit erneut zu starten.
  9. Klicken Sie auf Ja, um IBM Integration Toolkit erneut zu starten. Wenn die Ländereinstellung Ihres Betriebssystems für eine der Sprachen konfiguriert ist, die von den von Ihnen installierten Sprachenpaketen unterstützt werden, wird die IBM Integration Toolkit-Benutzerschnittstelle in dieser Sprache angezeigt.
