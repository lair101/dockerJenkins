IBM Integration Toolkit の各国語パックのインストール

症状:
バージョン 10 では、IBM Integration Bus のインストールに IBM Integration Toolkit の言語パックは組み込まれていません。デフォルトでは、IBM Integration Toolkit インターフェースは英語で表示されます。

解決策:
IBM Integration Toolkit Language Pack フィーチャーをダウンロードします。

IBM Integration Toolkit Language Pack フィーチャーには、以下の言語向けの言語パックが含まれています。

  • ドイツ語、スペイン語、フランス語、イタリア語、日本語、韓国語、ブラジル・ポルトガル語、中国語 (簡体字)、中国語 (繁体字) (Language Pack 1)
  • ポーランド語、ロシア語 (Language Pack 2)
  • トルコ語 (Language Pack 2a)

IBM Integration Toolkit Language Pack フィーチャーをインストールするには、以下のステップを実行します。

  1. Web ブラウザーで、次の URL にナビゲートします: ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. 該当する Language Pack フィーチャーをダウンロードします。
    注: ご使用の IBM Integration Bus のバージョンと一致する IBM Integration Toolkit Language Pack フィーチャーのバージョンを選択してください。
  3. IBM Integration Toolkit を開始します。
  4. メニューから、「ヘルプ」>「新規ソフトウェアのインストール」の順にクリックします。「使用可能なソフトウェア」ウィンドウが開きます。
  5. 「追加」をクリックし、リポジトリーの名前を入力します。例: Language Packs。
  6. 「アーカイブ」をクリックし、IBM Integration Toolkit Language Pack フィーチャーをダウンロードした場所にナビゲートし、「OK」をクリックします。IBM Integration Toolkit Language Pack フィーチャーが表示されます。
  7. IBM Integration Toolkit Language Pack フィーチャーを展開して、個々の言語パックを表示し、インストールする言語パックのチェック・ボックスを選択し、「次へ」をクリックします。
  8. 再度「次へ」をクリックし、ご使用条件の同意をクリックしてから「終了」をクリックします。選択した言語パックがインストールされ、IBM Integration Toolkit を再始動するようプロンプトが出されます。
  9. 「はい」をクリックして、IBM Integration Toolkit を再始動します。オペレーティング・システムのロケールが、インストールした言語パックによってサポートされるいずれかの言語に対応するように構成されていれば、IBM Integration Toolkit ユーザー・インターフェースはその言語で表示されます。
