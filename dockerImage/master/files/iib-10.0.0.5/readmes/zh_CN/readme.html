安装 IBM Integration Toolkit 的本地语言包

症状：
在 V10 中，IBM Integration Bus 安装不包含 IBM Integration Toolkit 的语言包。缺省情况下，IBM Integration Toolkit 界面显示为英语。

解决方案：
下载 IBM Integration Toolkit 语言包功能部件。

IBM Integration Toolkit 语言包功能部件包含以下语言的语言包：

  • 德语、西班牙语、法语、意大利语、日语、韩语、巴西葡萄牙语、简体中文、繁体中文（语言包 1）
  • 波兰语、俄语（语言包 2）
  • 土耳其语（语言包 2a）

要安装 IBM Integration Toolkit 语言包功能部件，请完成以下步骤：

  1. 在 Web 浏览器中，导航到以下 URL：ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. 下载相关语言包功能部件。
    注意：请确保选择与您的 IBM Integration Bus 版本匹配的 IBM Integration Toolkit 语言包功能部件版本。
  3. 启动 IBM Integration Toolkit。
  4. 从菜单中，单击帮助 > 安装新软件。将打开“可用软件”窗口。
  5. 单击添加，然后输入存储库的名称。例如：语言包。
  6. 单击归档，导航到下载了 IBM Integration Toolkit 语言包功能部件的位置，然后单击确定。此时将显示 IBM Integration Toolkit 语言包功能部件。
  7. 展开 IBM Integration Toolkit 语言包功能部件以显示各个语言包，选中要安装的语言包的复选框，然后单击下一步。
  8. 再次单击下一步，单击以接受许可协议，然后单击完成。此时将安装所选语言包，并提示您重新启动 IBM Integration Toolkit。
  9. 单击是以重新启动 IBM Integration Toolkit。如果针对所安装的语言包支持的其中一种语言配置了操作系统语言环境，那么 IBM Integration Toolkit 用户界面显示为该语言。
