Installation des modules de langue nationale d'IBM Integration Toolkit

Symptôme :
Dans la version 10, l'installation d'IBM Integration Bus n'inclut pas les modules de langue d'IBM Integration Toolkit. Par défaut, l'interface d'IBM Integration Toolkit est affichée en anglais.

Solution :
Téléchargez la fonctionnalité des modules de langue IBM Integration Toolkit.

La fonctionnalité des modules de langue IBM Integration Toolkit contient les modules de langue des langues suivantes :

  • Allemand, espagnol, français, italien, japonais, coréen, portugais (Brésil), chinois simplifié, chinois traditionnel (module de langue 1)
  • Polonais, russe (module de langue 2)
  • Turc (module de langue 2a)

Pour installer une fonctionnalité de module de langue d'IBM Integration Toolkit, procédez comme suit :

  1. Dans un navigateur Web, accédez à l'URL suivante : ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. Téléchargez la fonctionnalité de module de langue appropriée.
    Remarque : sélectionnez bien la version de la fonctionnalité des modules de langue IBM Integration Toolkit qui correspond à votre version d'IBM Integration Bus.
  3. Démarrez IBM Integration Toolkit.
  4. Dans le menu, cliquez sur Aide > Installer un nouveau logiciel. La fenêtre Logiciels disponibles s'ouvre.
  5. Cliquez sur Ajouter et entrez un nom pour le référentiel. Par exemple : Modules de langue.
  6. Cliquez sur Archiver, accédez à l'emplacement où vous avez téléchargé la fonctionnalité des modules de langue IBM Integration Toolkit, puis cliquez sur OK. La fonctionnalité des modules de langue IBM Integration Toolkit est affichée.
  7. Développez la fonctionnalité des modules de langue IBM Integration Toolkit pour afficher les modules de langue individuels, cochez les cases des modules de langue à installer, puis cliquez sur suivant.
  8. Cliquez de nouveau sur Suivant, cliquez pour accepter le contrat de licence, puis cliquez sur Terminer. Les modules de langue sélectionnés sont installés et vous êtes invité à redémarrer IBM Integration Toolkit.
  9. Cliquez sur Oui pour redémarrer IBM Integration Toolkit. Si l'environnement local de votre système d'exploitation est configuré pour l'une des langues prises en charge par les modules de langue que vous avez installés, l'interface utilisateur d'IBM Integration Toolkit est affichée dans cette langue.
