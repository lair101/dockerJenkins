Установка языковых пакетов для IBM Integration Toolkit

Признак:
В версии 10 установка IBM Integration Bus не включает языковые пакеты для IBM Integration Toolkit. По умолчанию в интерфейсе IBM Integration Toolkit применяется английский язык.

Решение:
Загрузите компонент IBM Integration Toolkit Language Pack.

Компонент IBM Integration Toolkit Language Pack содержит языковые пакеты для следующих языков:

  • Немецкий, испанский, французский, итальянский, японский, корейский, португальский (Бразилия), китайский (Тайвань) и китайский (КНР) - языковый пакет 1
  • Польский, русский - языковый пакет 2
  • Турецкий - языковый пакет 2a

Для установки компонента IBM Integration Toolkit Language Pack выполните следующие действия:

  1. В веб-браузере откройте следующий URL: ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. Загрузите требуемый компонент Language Pack.
    Примечание: выберите версию компонента IBM Integration Toolkit Language Pack, совпадающую с версией IBM Integration Bus.
  3. Запустите IBM Integration Toolkit.
  4. В меню выберите Справка > Установить новое программное обеспечение. Откроется окно Доступное программное обеспечение.
  5. Нажмите кнопку Добавить и введите имя хранилища. Пример: Language Packs.
  6. Выберите Архив, перейдите в расположение, в которое был загружен компонент IBM Integration Toolkit Language Pack, и нажмите кнопку OK. Отображается компонент IBM Integration Toolkit Language Pack.
  7. Разверните компонент IBM Integration Toolkit Language Pack для отображения отдельных языковых пакетов, включите переключатели рядом с требуемыми языковыми пакетами и нажмите кнопку Далее.
  8. Снова нажмите кнопку Далее, примите лицензионное соглашение и нажмите кнопку Готово. Выбранные языковые пакеты устанавливаются и предлагается перезапустить IBM Integration Toolkit.
  9. Нажмите кнопку Да, чтобы перезапустить IBM Integration Toolkit. Если локаль операционной системы настроена для одного из языков, поддерживаемых установленными языковыми пакетами, то пользовательский интерфейс IBM Integration Toolkit отображается на этом языке.
