IBM Integration Toolkit için ulusal dil paketlerinin kurulması

Belirti:
Sürüm 10'da, IBM Integration Bus kuruluşu IBM Integration Toolkit'e ilişkin dil paketlerini içermez. Varsayılan olarak, IBM Integration Toolkit arabirimi İngilizce görüntülenir.

Çözüm:
IBM Integration Toolkit Language Pack adlı özelliği yükleyin.

IBM Integration Toolkit Language Pack özelliği aşağıdaki dillere ilişkin dil paketlerini içerir:

  • Almanca, İspanyolca, Fransızca, İtalyanca, Japonca, Korece, Brezilya Portekizcesi, Yalınlaştırılmış Çince, Geleneksel Çince (Language Pack 1)
  • Polonya dili, Rusça (Dil Paketi 2)
  • Türkçe (Dil Paketi 2a)

Bir IBM Integration Toolkit Language Pack özelliğini kurmak için aşağıdaki adımları izleyin:

  1. Bir web tarayıcısında şu URL adresine gidin: ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. İlgili Language Pack özelliğini karşıdan yükleyin.
    Not: IBM Integration Toolkit Language Pack özelliğinin IBM Integration Bus sürümünüzle eşleşen sürümünü seçtiğinizden emin olun.
  3. IBM Integration Toolkit'i başlatın.
  4. Menüden Help > Install New Software (Yardım - Yeni Yazılım Kur) öğelerini seçin. Available Software (Kullanılabilecek Yazılımlar) penceresi açılır.
  5. Add (Ekle) düğmesini tıklatın ve havuz için bir ad girin. Örnek: Language Packs.
  6. Archive (Arşivle) düğmesini tıklatın ve IBM Integration Toolkit Language Pack özelliğini yüklediğiniz yere gidip OK (Tamam) düğmesini tıklatın. IBM Integration Toolkit Language Pack özelliği görüntülenir.
  7. IBM Integration Toolkit Language Pack özelliğini genişleterek her bir dil paketini görüntüleyin, kurmak istediğiniz dil paketlerine ilişkin onay kutularını seçin ve Next (Sonraki) düğmesini tıklatın.
  8. Next düğmesini yeniden tıklatın, lisans sözleşmesini kabul edin ve Finish (Son) düğmesini tıklatın. Seçilen dil paketleri kurulur ve IBM Integration Toolkit'i yeniden başlatmanız istenir.
  9. Yes (Evet) düğmesini tıklatarak IBM Integration Toolkit'i yeniden başlatın. İşletim sisteminizin ülke değeri, kurduğunuz dil paketlerinin desteklediği dillerden biri için yapılandırıldıysa, IBM Integration Toolkit kullanıcı arabirimi o dilde görüntülenir.
