IBM Integration Toolkit의 자국어(NL)팩 설치

증상:
버전 10에서, IBM Integration Bus 설치에는 IBM Integration Toolkit의 언어팩이 포함되지 않습니다. 기본적으로 IBM Integration Toolkit 인터페이스는 영어로 표시됩니다.

솔루션:
IBM Integration Toolkit 언어팩 기능을 다운로드하십시오.

IBM Integration Toolkit 언어팩 기능에는 다음 언어에 대한 언어팩이 포함됩니다.

  • 독일어, 스페인어, 프랑스어, 이탈리아어, 일본어, 한국어, 브라질식 포르투갈어, 중국어, 대만어(언어팩 1)
  • 폴란드어, 러시아어(언어팩 2)
  • 터키어(언어팩 2a)

IBM Integration Toolkit 언어팩 기능을 설치하려면 다음 단계를 완료하십시오.

  1. 웹 브라우저에서 다음 URL로 이동하십시오. ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. 관련 언어팩 기능을 다운로드하십시오.
    참고: IBM Integration Bus의 버전과 일치하는 IBM Integration Toolkit 언어팩 기능의 버전을 선택해야 합니다.
  3. IBM Integration Toolkit을 시작하십시오.
  4. 메뉴에서 도움말 > 새 소프트웨어 설치를 클릭하십시오. 사용 가능 소프트웨어 창이 열립니다.
  5. 추가를 클릭하고 저장소의 이름을 입력하십시오. 예를 들어 Language Pack입니다.
  6. 아카이브를 클릭하고 IBM Integration Toolkit 언어팩 기능을 다운로드한 위치로 이동하고 확인을 클릭하십시오. IBM Integration Toolkit 언어팩 기능이 표시됩니다.
  7. IBM Integration Toolkit 언어팩 기능을 펼쳐 개별 언어팩을 표시하고 설치할 언어팩의 선택란을 선택하고 다음을 클릭하십시오.
  8. 다음을 다시 클릭하고 라이센스 계약에 동의하도록 클릭한 후 완료를 클릭하십시오. 선택한 언어팩이 설치되고 IBM Integration Toolkit을 재시작하도록 프롬프트됩니다.
  9. 예를 클릭하여 IBM Integration Toolkit을 재시작하십시오. 운영 체제 로케일이 설치한 언어팩이 지원하는 언어 중 하나에 대해 구성되면 IBM Integration Toolkit 사용자 인터페이스가 해당 언어로 표시됩니다.
