安裝 IBM Integration Toolkit 的國家語言套件

症狀：
在第 10 版中，IBM Integration Bus 的安裝不包含 IBM Integration Toolkit 的語言套件。依預設，IBM Integration Toolkit 介面以英文顯示。

解決方案：
下載 IBM Integration Toolkit Language Pack 功能。

IBM Integration Toolkit Language Pack 功能包含下列語言的語言套件：

  • 德文、西班牙文、法文、義大利文、日文、韓文、巴西葡萄牙文、簡體中文、繁體中文（語言套件 1）
  • 波蘭文、俄文（語言套件 2）
  • 土耳其文（語言套件 2a）

若要安裝 IBM Integration Toolkit Language Pack 功能，請完成下列步驟：

  1. 在 Web 瀏覽器中，導覽至下列 URL： ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. 下載相關的語言套件功能。
    附註：請確定選取符合您的 IBM Integration Bus 版本的 IBM Integration Toolkit Language Pack 功能版本。
  3. 啟動 IBM Integration Toolkit。
  4. 在功能表上，按一下說明 > 安裝新軟體。這時會開啟「可用的軟體」視窗。
  5. 按一下新增並輸入儲存庫的名稱。例如：Language Packs。
  6. 按一下保存，導覽至 IBM Integration Toolkit Language Pack 功能的下載位置，然後按一下確定。此時顯示 IBM Integration Toolkit Language Pack 功能。
  7. 展開 IBM Integration Toolkit Language Pack 功能以顯示個別語言套件，選取您要安裝的語言套件的勾選框，然後按下一步。
  8. 再按下一步，按一下以接受授權合約，然後按一下完成。所選的語言套件已安裝，並提示您重新啟動 IBM Integration Toolkit。
  9. 按一下是，以重新啟動 IBM Integration Toolkit。如果您的作業系統語言環境配置為您所安裝的語言套件支援的某種語言，則 IBM Integration Toolkit 使用者介面會以該語言顯示。
