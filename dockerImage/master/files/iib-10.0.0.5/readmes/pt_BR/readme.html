Instalando os pacotes de idioma nacional para o IBM Integration Toolkit

Sintoma:
Na versão 10, a instalação do IBM Integration Bus não inclui pacotes de idioma para o IBM Integration Toolkit. Por padrão, a interface do IBM Integration Toolkit é exibida em inglês.

Solução:
Faça o download do recurso IBM Integration Toolkit Language Pack.

O recurso IBM Integration Toolkit Language Pack contém pacotes de idioma para os idiomas seguir:

  • Alemão, espanhol, francês, italiano, japonês, coreano, português brasileiro, chinês simplificado, chinês tradicional (Language Pack 1)
  • Polonês, russo (Language Pack 2)
  • Turco (Language Pack 2a)

Para instalar um recurso do IBM Integration Toolkit Language Pack, conclua as seguintes etapas:

  1. Em um navegador da web, acesse a URL a seguir:ftp://public.dhe.ibm.com/software/integration/integrationbus/nl/
  2. Faça o download do recurso Language Pack relevante.
    Nota: assegure-se de que selecionou a versão do recurso IBM Integration Toolkit Language Pack que corresponde a versão do IBM Integration Bus.
  3. Inicie o IBM Integration Toolkit.
  4. No menu, clique em Ajuda > Instalar novo software. A janela Software disponível é aberta.
  5. Clique em Incluir e insira um nome para o repositório. Por exemplo: Pacotes de idiomas.
  6. Clique em Arquivar, navegue até o local em que fez o download do recurso IBM Integration Toolkit Language Pack e clique em OK. O recurso IBM Integration Toolkit Language Pack é exibido.
  7. Expanda o recurso IBM Integration Toolkit Language Pack para exibir os pacotes de idioma individuais, selecione as caixas de seleção para os pacotes de idioma que deseja instalar e clique em Avançar.
  8. Clique em Avançar novamente, clique para aceitar o contrato de licença e, em seguida, clique em Concluir. Os pacotes de idiomas selecionados são instalados e é solicitado que você reinicie o IBM Integration Toolkit.
  9. Clique em Sim para reiniciar o IBM Integration Toolkit. Se o código de idioma do sistema operacional estiver configurado para um dos idiomas que é suportado pelos pacotes de idioma que você instalou, a interface com o usuário do IBM Integration Toolkit é exibida nesse idioma.
